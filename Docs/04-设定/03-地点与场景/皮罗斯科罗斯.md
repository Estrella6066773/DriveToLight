# 皮罗斯科罗斯（循烬城，Pyroskóros）

## 摘要

玩家控制的移动城市，意为「逐火之蛾」。前文明遗留的高级载具，玩家（城主/舰长）驾驶它追逐太阳、延续文明。可能是第三颗骄阳之心的所在地。

## 名称释义

> **引用原文（保留）**
>
> "玩家主城的名字：Pyroskóros（皮罗斯科罗斯，循烬城），意为逐火之蛾。"

**Pyroskóros**（希腊语词根组合）：
- **Pyr** = 火（pyro-）
- **Skóros** = 追踪者、追随者（scoros）

中文译名「循烬城」：
- **循** = 追逐、遵循
- **烬** = 火焰的余烬、太阳的光芒
- **城** = 移动城市

**寓意**：
- 如同飞蛾追逐火焰，这座城市追逐着天穹上的骄阳。
- 隐喻：它追逐的「太阳」并非自然之火，而是人造的余烬——正如它自己，也是前文明遗留的「余烬」。
