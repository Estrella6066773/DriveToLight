# 地点与场景

本目录存放《循光之城》的**地点设定、场景描述、地理关系**。

> 对外读者请先从 [游戏介绍](../../游戏介绍.md) 入手；本目录各文档为地理与地点专题详解。

## 地点总览

### 核心地理节点（按章节出场顺序）

| 地点 | 类型 | 出现章节 | 说明 |
|------|------|----------|------|
| [日生之地](日生之地.md) | 安全区 | 第一章 | 玩家起始点，新文明的聚居地 |
| [铁门关](铁门关.md) | 关卡/边界 | 第一-二章 | 日生之地的边界门户 |
| [荒地](荒地.md) | 荒野区域 | 第一章-第二章 | 太阳城边境外的危险区域 |
| [铁壳](../02-角色与势力/铁壳.md) | 势力-机械体单位 | 第二章 | 游荡于铁门关之外的机械体，与无敌骄阳会敌对，击败掉落建材与燃料 |
| [太阳城（骄阳之城）](太阳城.md) | 外部城市 | 第一章 | 位于太阳正下方的强大城市，拥有第二颗骄阳之心 |

| [指挥塔](指挥塔.md) | 控制中枢 | 第五章 | 方舟控制中心，位于暗渊深处（详见 [05-隐秘真相/指挥塔的真相](../05-隐秘真相/指挥塔的真相.md)） |
