## 地图地理关系

> **结构说明（仅此节使用「卷轴」）**：世界地图为纵向六边形**卷轴**（程序见 [地图与移动](../../02-系统设计/02-地图与世界/地图与移动.md)）。下列命名地区在地图上**自下而上**排列；**其它文档不写「卷轴」**，只用向上/向下、主线上下方向等说法。

| 顺序 | 地区 | 说明 |
|------|------|------|
| 1（底） | [渊光](./渊光.md) | 暗渊最深处的高塔 |
| 2 | [渊光城](./渊光城.md) | 塔脚下的传说城市 |
| 3 | [暗渊](./暗渊.md) | **深处段**：渊光城之上、玩家起点之下 |
| 4 | [玩家起点](./玩家起点.md) | 玩家苏醒并建立循烬城的开局区段 |
| 5 | [日生之地](./日生之地.md) | 日生文明主要聚居区 |
| 6 | [铁门关](./铁门关.md) | 聚居区与荒地的唯一通道 |

| 8 | [暗渊](./暗渊.md) | **太阳经过段**：太阳固定时为暗渊；太阳向关外移动时暂被照亮 |

| 10（顶） | [铁巢](./铁巢.md) | 本地区中枢（IronNest） |

**暗渊**指**一切不被太阳照射的区域**——民众对「黑暗 / 危险」的唯一说法。纵向主线上有**两处命名区段**（深处段、太阳经过段），玩法上均属暗渊；其它无日照格子亦按同一规则处理，详见 [暗渊](./暗渊.md)。

**日照与地区名分离**：第一、二章追日，循烬城仍在太阳照射范围内。第三章起**太阳已离开循烬城行进路线**——此后无论途经 [荒地](./荒地.md)、[铁门关](./铁门关.md)、[日生之地](./日生之地.md) 等**何地名**，当时**均无日照**，环境上**一律属暗渊**。

**行进方向**

- **追日**（第一、二章）：**向上**，有日照；太阳自固定位置起**沿固定方向**（向关外）移动，循烬城**同向**追赶。光随太阳进入上方地区。玩家起点 → 日生之地 → 铁门关 → 荒地 → 暗渊（太阳经过段，此时被照亮）→ 铁巢废墟 → 铁巢。第二章难点是**太阳加速、速度差拉大**，不是路线难测。
- **入暗**（第三章）：太阳最后余晖离去，向暗渊冲锋；**向下**，全程无日照。
- **暗途**（第四章）：仍无日照，**向下**，途经铁门关、日生之地（地理名）至玩家起点，沿途救援与道德抉择。
- **终局**（第五章）：**向下**，抵达暗渊最深处（深处段）的 [渊光](./渊光.md) / [渊光城](./渊光城.md)。

叙事行进与章节节奏见 [章节划分与故事大纲](../05-隐秘真相/章节划分与故事大纲.md#章节总览)。

### 章节与行进方向

| 章节 | 名称 | 方向 | 日照 | 主要地区 |
|------|------|------|------|----------|
| 第一章 | 初速度 | 向上 | 有 | 玩家起点 → 日生之地 → 太阳城（边境）→ 铁门关 |
| 第二章 | 角速度 | 向上 | 有 | 铁门关 → 荒地 → 暗渊（太阳经过段）→ 铁巢废墟 → 铁巢 |
| 第三章 | 离心力 | 向下 | **无** | 铁巢 → 铁巢废墟 → 暗渊（太阳经过段）→ 荒地 → 铁门关 / 日生之地 |
| 第四章 | 摩擦力 | 向下 | **无** | 铁门关 → 日生之地 → 玩家起点 |
| 第五章 | 向心力 | 向下 | **无** | 玩家起点 → 暗渊（深处段）→ 渊光城 / 渊光 |
