# 矿区

## 摘要

荒野中的资源点，可开采建材与燃料。新文明的重要资源来源，也是驱动移动城市运转的基础。

## 基本信息

| 属性 | 内容 |
|------|------|
| 名称 | 矿区 |
| 类型 | 荒野资源点 |
| 产出 | 建材、燃料 |
| 分布 | [荒地](./荒地.md)各处 |

## 玩家可见层

### 描述

- 荒野中的矿物开采点。
- 可以建立采集站，持续产出建材与燃料。
- 由[工程队](../../02-系统设计/01-核心系统/队伍系统.md)开发和运营。

### 与玩法的关系

- 四种核心资源之一的重要来源。
- 建材用于建造/修复城区、设施。
- 燃料用于驱动城市移动、维持能源。
- 需要在城市停下时派出工程队开发。

## 设定母本（内部）

### 方舟结构中的位置

- 分布于环状方舟内表面的[荒地](./荒地.md)区域。
