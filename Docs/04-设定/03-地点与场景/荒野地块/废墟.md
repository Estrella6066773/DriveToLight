# 废墟

## 摘要

**建筑层**上**状态为废墟的城区**：损毁的城区实例，**不是**与「城区」并列的类型，**不是**资源层 [遗迹](./遗迹.md)（前文明遗迹、不可重建）。

## 玩家可见

曾有人居住或运转过的**城市残骸**；地图与信息层须让玩家**能认出**这是**废墟**，而非状态正常的城区或资源层遗迹。地图可**默认刷新**废墟，亦可在战斗 / 事件中将正常城区击毁为此状态。

**不可迁入**新居民；已在该城区的居民**仅可迁出**。**无法工作**。须花费资源并完成修复，将**结构完整度**恢复至 **20% 及以上**且状态变为**正常**后，才**视为**正常城区（修复细则 **待定**）。

## 玩法关联

- 规则见 [城市模块化 · 废墟](../../../02-系统设计/03-模块与城市/城市模块化.md#废墟)、[荒野地点 · 遗迹与废墟](../../../02-系统设计/02-资源循环/荒野地点.md#遗迹与废墟术语强制区分)。
- 命名地区 [铁巢废墟](../铁巢废墟.md) **不是** 本词条（网格城区状态）。
