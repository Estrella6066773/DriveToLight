# 废墟

## 摘要

荒野中的遗迹，可搜刮建材、燃料、食物。前文明时代的遗留，可能藏有关于方舟结构和人造太阳的线索。

## 基本信息

| 属性 | 内容 |
|------|------|
| 名称 | 废墟 |
| 类型 | 荒野遗迹/资源点 |
| 产出 | 建材、燃料、食物 |
| 分布 | [荒地](./荒地.md)各处 |
| 特殊价值 | 可能藏有前文明线索 |

## 玩家可见层

### 描述

- 荒野中散落的遗迹建筑，风格与当前新文明截然不同。
- 可以搜刮多种资源：建材、燃料、食物。
- 规模大小不一，暗示着「曾经繁华的过去」。
- 新文明居民无法理解这些建筑的技术与用途。

### 与玩法的关系

- 综合性资源点，可获取多种资源。
- 某些大型废墟（如[铁巢废墟](./铁巢废墟.md)）藏有关键线索。
- 探索废墟是了解「过去」的窗口。

## 设定母本（内部）

### 方舟结构中的位置

- 环状方舟内表面，前文明时代的建筑遗迹。
- 天体撞击与太阳停转导致这些区域被废弃，变成废墟。
- 新文明的聚居地（日生之地）也是在前文明废墟上重建的。

### 线索价值

| 废墟类型 | 可能线索 |
|----------|----------|
| 普通废墟 | 前文明生活用品，暗示生活方式 |
| 工业废墟 | 前文明技术痕迹 |
| 铁巢废墟 | 方舟结构、人造太阳的关键线索 |
