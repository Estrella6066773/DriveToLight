# 果地

## 摘要

荒野中的资源点，可采集食物。在资源紧缺的荒地中，果地是维持人口生存的重要补给来源。

## 基本信息

| 属性 | 内容 |
|------|------|
| 名称 | 果地 |
| 类型 | 荒野资源点 |
| 产出 | 食物 |
| 分布 | [荒地](./荒地.md)各处 |

## 玩家可见层

### 描述

- 荒野中能够产出食物的自然或半自然区域。
- 可能包括：可采集的野果林、菌菇地、小型农场遗迹等。
- 由[工程队](../../02-系统设计/01-核心系统/队伍系统.md)开发运营，产出食物。

### 与玩法的关系

- 四种核心资源之一的重要来源。
- 食物用于维持人口消耗、远征队补给。
- 在荒地深处资源稀缺，果地尤为珍贵。

## 设定母本（内部）

### 方舟结构中的位置

- 分布于环状方舟内表面的[荒地](./荒地.md)区域。
- 可能是前文明时代的农业设施遗迹，或自然环境中的可食用植物区。
