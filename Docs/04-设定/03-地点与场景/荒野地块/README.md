# 荒野地块

← [地点与场景](../README.md)

荒野网格上的**可刷新地点类型**。玩法规则见 [荒野地点](../../../02-系统设计/02-资源循环/荒野地点.md)；分布区域见 [荒地](../荒地.md)。

## 类型一览

| 类型 | 主要产出 | 简述 |
|------|----------|------|
| [矿区](./矿区.md) | 建材、燃料 | 矿物开采点，由工程队开发 |
| [果地](./果地.md) | 食物 | 可采集食物的野地或农业遗迹 |
| [废墟](./废墟.md) | 建材、燃料、食物 | 前文明遗迹，可搜刮多种资源 |
| [村镇](./村镇.md) | 食物、人口 | 小型定居点 |
| [城市-外部](./城市-外部.md) | 建材、燃料、食物、人口 | 独立外部城市；重要特例：[太阳城](../太阳城.md) |

> **区分**：[废墟](./废墟.md) 是荒野地块类型；[铁巢废墟](../铁巢废墟.md) 是命名**地区**（本质为巨大前文明构造体），非普通废墟带。
