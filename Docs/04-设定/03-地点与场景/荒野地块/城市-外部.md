# 城市-外部

## 摘要

荒野中的外部城市，与玩家的移动城市不同，它们是固定据点（或不可移动）。可提供建材、燃料、食物、人口，拥有独立的关系系统。

> 注：[太阳城（骄阳之城）](./太阳城.md) 是最重要的外部城市，有独立词条。

## 基本信息

| 属性 | 内容 |
|------|------|
| 名称 | 城市（外部城市） |
| 类型 | 外部城市/固定据点 |
| 产出 | 建材、燃料、食物、人口 |
| 分布 | [荒地](./荒地.md)各处，[日生之地](./日生之地.md)内 |
| 性质 | 固定据点，不可被工程队开采 |
| 特殊 | 拥有独立的关系系统 |

## 玩家可见层

### 描述

- 荒野中分布的城市，与玩家的移动城市不同，它们**不可整体移动**（或移动方式不同）。
- 规模通常比[村镇](./村镇.md)大，拥有更复杂的社会结构。
- 可以提供各种资源，也可招募人口。
- 每座城市拥有独立的关系系统，可以建立友好或敌对关系。
- **默认不能被[工程队](../../02-系统设计/01-核心系统/队伍系统.md)开采**。

### 与玩法的关系

- 资源、人口的重要来源。
- 关系系统的核心对象：与每座城市建立关系，影响互动选项。
- 组织系统：某些城市归属于同一组织，关系变化会传导。
- 第四章中，沿途城市处于濒危状态，需要做出救援抉择。

## 设定母本（内部）

### 方舟结构中的位置

- 分布于环状方舟内表面。
- [日生之地](./日生之地.md)内有较稳定的城市。
- [荒地](./荒地.md)中也有城市分布，但环境更恶劣，生存更艰难。

### 特殊案例：太阳城（骄阳之城）

- [太阳城](./太阳城.md)是最重要的外部城市，位于太阳正下方。
- 拥有[骄阳之心](../05-隐秘真相/骄阳之心.md)，能源近乎无限。
- 由骄阳之子统治，与玩家有复杂的关系。
