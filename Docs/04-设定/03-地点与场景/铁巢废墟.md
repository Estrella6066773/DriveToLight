# 铁巢废墟（IronWastes）

## 摘要

荒地深处的废弃**地区**（IronWastes），遍布前文明城市残骸。深处的 [铁巢（IronNest）](./铁巢.md) 是 [铁壳](../02-角色与势力/铁壳.md) 总部所在。

## 玩家可见

- 规模远超普通遗迹的前文明废墟带；环境比 [荒地](./荒地.md) 更恶劣。
- 太阳移动前曾与 [暗渊](./暗渊.md) 并列；此后被黑暗笼罩的区域渐被统称暗渊。
- 废墟深处有一处仍在运转的前文明城市——即 **铁巢**。

## 玩法关联

- **第二章**：穿越铁巢废墟地区，遭遇铁壳巡逻，终点抵达 [铁巢](./铁巢.md)。
- **第三章起点**：在铁巢或废墟边缘完成准备后向暗渊冲锋。
- L1 方舟结构线索见 [世界的真相](../05-隐秘真相/世界的真相.md)；章节见 [章节划分与故事大纲](../05-隐秘真相/章节划分与故事大纲.md)。
