# 暗渊

## 摘要

太阳照射不到的区域——**凡不被太阳照射之处，居民统称暗渊**。纵向主链上有**两处命名区段**（深处段、太阳经过段），居民不作区分；其它无日照格子亦属暗渊。

## 玩家可见

- 常识：无人主动进入暗渊；传说深处有旧神或先祖遗迹。
- 无「黑夜」概念：太阳变暗称「黄昏」或「长暗季」；照不到的地方统称暗渊。
- 环境寒冷、资源匮乏，居民将其恐怖化（见 [太阳的真相](../05-隐秘真相/太阳的真相.md)）。
- **太阳经过段**：太阳仍在天顶附近经过，地面却得不到光照——居民对此缺乏解释，只觉关外更险。

## 玩法关联

- **第三至五章**：太阳已离开主城路线，**全程无日照**；向下，地理上可途经铁门关、日生之地等，**环境上均属暗渊**。
- **第五章**：抵达 [暗渊](./暗渊.md) **深处段**（最深处）的 [渊光](./渊光.md) / [渊光城](./渊光城.md)——非首次进入暗渊。
- 地图地理与章节见 [地点与场景 · 地图地理关系](./README.md#地图地理关系)、[章节划分与故事大纲](../05-隐秘真相/章节划分与故事大纲.md)。
