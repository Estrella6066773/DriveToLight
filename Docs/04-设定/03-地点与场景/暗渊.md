# 暗渊

## 摘要

太阳照射不到的区域——**凡不被太阳照射之处，民众统称暗渊**。纵向主线上有**两处文档分区**（深处段、太阳经过段），民众不作区分；其它无日照格子亦属暗渊。

**太阳曾长期停在固定位置**（见 [世界的真相](../05-隐秘真相/世界的真相.md)）；**剧情中**太阳才开始移动。两处区段的日照随太阳是否照到、是否正在移离而变化，见下节「太阳经过段」。

## 玩家可见

- 常识：无人主动进入暗渊；**世人**传说深处有旧神或先祖遗迹。
- **用语**：照不到太阳之处统称**暗渊**；太阳暗淡之时称**黄昏**。民众无「黑夜」这一说法。
- 环境寒冷、资源匮乏、寸草不生，民众将其恐怖化（见 [太阳的真相](../05-隐秘真相/太阳的真相.md)）。
- **太阳经过段**（文档分区名，荒地与铁巢废墟之间）：太阳**固定时**此处终年无光，与关外一样属暗渊。**太阳开始移动、向关外（向上）离去时**，光随太阳移入此方向，本地区才被照亮——**仅因太阳往此方向移动**，不是“太阳在头顶却照不到地面”。第一、二章追日即紧跟这道离去的光；第三起太阳再次远离，此处重归暗渊。

## 玩法关联

- **第三至五章**：太阳已离开循烬城路线，**全程无日照**；向下，地理上可途经铁门关、日生之地等，**环境上均属暗渊**。
- **第五章**：抵达 [暗渊](./暗渊.md) **深处段**（最深处）的 [渊光](./渊光.md) / [渊光城](./渊光城.md)——非首次进入暗渊。
