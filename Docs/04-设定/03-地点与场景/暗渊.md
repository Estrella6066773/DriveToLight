# 暗渊

## 摘要

太阳照射不到的区域——新文明居民对「黑暗/危险」的**唯一说法**。他们从未见过真正的黑夜。

## 玩家可见

- 常识：无人主动进入暗渊；传说深处有旧神或先祖遗迹。
- 无「黑夜」概念：太阳变暗称「黄昏」或「长暗季」；照不到的地方统称暗渊。
- 环境寒冷、资源匮乏，居民将其恐怖化（见 [太阳的真相](../05-隐秘真相/太阳的真相.md)）。

## 玩法关联

- **第三章**：背离太阳，向暗渊冲锋。
- **第四章**：穿越暗渊返回另一侧。
- **第五章**：抵达深处的 [渊光](./渊光.md) / [渊光城](./渊光城.md)。
- 章节节奏见 [章节划分与故事大纲](../05-隐秘真相/章节划分与故事大纲.md)。
