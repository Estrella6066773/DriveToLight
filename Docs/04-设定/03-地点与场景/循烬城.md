# 循烬城（皮罗斯科罗斯，Pyroskóros）





## 摘要



玩家移动城市。**循烬城**为中文名；**皮罗斯科罗斯**（Pyroskóros）为其音译，意为「逐火之蛾」——**二者指同一座城市**。世上**第一座且唯一一座**可整城迁徙的前文明载具；核心区反应堆专名 **Pyrois**（皮洛伊斯，见 [骄阳之心](../05-隐秘真相/骄阳之心.md)）。剧情推进后接入、随城迁徙的城区，亦属循烬城。



## 玩家可见



- 由可拆分重组的城区组成；在荒野移动、停下、建设、探索。

- 民众与外力多称**循烬城**；文献与城主口语中亦可见 **皮罗斯科罗斯** / **Pyroskóros**。

- 城主带领幸存者追逐离去的太阳以延续文明。

- 城主对城市技术异常熟悉（前文明指挥官知识的伏笔）。
