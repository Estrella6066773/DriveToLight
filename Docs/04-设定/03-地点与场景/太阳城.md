# 太阳城（骄阳之城）

## 摘要

位于太阳正下方的强大外部城市，由 [无敌骄阳会](../02-角色与势力/无敌骄阳会.md) 与骄阳之子统治。拥有**第三颗骄阳之心**（母本见 [骄阳之心](../05-隐秘真相/骄阳之心.md)）。

## 玩家可见

- 环境温和、能源充裕，城市繁荣。
- 与 [铁壳](../02-角色与势力/铁壳.md) 在 [荒地](./荒地.md) 持续敌对。
- 与玩家主城关系复杂：既有冲突，终局亦可能需合作。

## 玩法关联

- **第一章**：边境冲突；骄阳之子点明循烬城拥有第一颗骄阳之心后放行。
- **第四章**：[铁门关](./铁门关.md) 冲突与放行。
- **第五章**：第三颗骄阳之心与终局相关（见 [指挥塔的真相](../05-隐秘真相/指挥塔的真相.md)）。
