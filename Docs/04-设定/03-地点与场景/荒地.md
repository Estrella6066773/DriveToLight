# 荒地

## 摘要

[日生之地](./日生之地.md) 之外的荒野。资源并不贫瘠，但因 [铁壳](../02-角色与势力/铁壳.md) 活动而极危险，是追逐太阳的必经之路。

## 玩家可见

- 比驻地更严酷；太阳轨迹难预测，追赶窗口有限。
- 散布 [荒野地块](./荒野地块/README.md) 各类地点与前文明残骸。
- 第二章可见对象主要为 [无敌骄阳会](../02-角色与势力/无敌骄阳会.md) 猎壳人与 [铁壳](../02-角色与势力/铁壳.md) 单位。

## 玩法关联

- 规则见 [荒野地点](../../02-系统设计/02-资源循环/荒野地点.md)。
- 经 [铁门关](./铁门关.md) 进入；深处连接 [铁巢废墟](./铁巢废墟.md)、[暗渊](./暗渊.md)。
- 章节范围见 [章节划分与故事大纲](../05-隐秘真相/章节划分与故事大纲.md)。
