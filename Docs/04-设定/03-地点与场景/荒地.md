# 荒地

## 摘要

[日生之地](./日生之地.md) 之外的荒野。资源并不贫瘠，但因 [铁壳](../02-角色与势力/铁壳.md) 活动而极危险，是追逐太阳的必经之路。

## 玩家可见

- 比驻地更严酷；太阳已自长期固定位置起**沿固定方向**（向关外）移动，玩家追日亦**沿同一方向**向上。制约来自太阳越转越快、城市速度有限，**不是**路线难测。
- 散布 [荒野地块](./荒野地块/README.md) 各类地点与前文明残骸。
- **第二章前段**：仅见 [猎壳人](../02-角色与势力/猎壳人.md) 营地与各式 [铁壳](../02-角色与势力/铁壳.md)；二者相爱相杀，玩家为继续前进须与之打交道（站队见 [章节划分与故事大纲](../05-隐秘真相/章节划分与故事大纲.md)）。
