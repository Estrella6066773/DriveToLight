# 日生之地

## 摘要

纵向主链上的**安全聚居区**（**第一、二章**太阳仍照及时）：日生文明重建后的主要活动地带。与 [玩家起点](./玩家起点.md) 相邻，范围更大。**第三至五章太阳已离去，途经本地区时亦无日照**，见 [暗渊](./暗渊.md)。

## 玩家可见

- 新文明在此重建；环境相对温和，有村镇与城市分布。
- 经 [铁门关（IronGate）](./铁门关.md) 与 [荒地](./荒地.md) 相连；关外视为未知与危险。
- 无「黑夜」概念；太阳变暗称「黄昏」或「长暗季」，照不到处统称 [暗渊](./暗渊.md)。

## 玩法关联

- **第一章**：自 [玩家起点](./玩家起点.md) 进入，穿越本地区向 [铁门关](./铁门关.md) 进发（**有日照**）。
- **第四章**：向下途经本地区，**无日照**；沿途救援濒危城市。
- [太阳城（骄阳之城）](./太阳城.md) 横向分布于追日路径一带，与本地区有贸易/冲突关系。
