# 日生之地

## 摘要

纵向主线上的**安全聚居区**（**第一、二章**太阳仍照及时）：日生文明重建后的主要活动地带；**诸王国、诸城邦**皆分布于此。与 [玩家起点](./玩家起点.md) 相邻，范围更大。**第三至五章太阳已离去，途经本地区时亦无日照**，见 [暗渊](./暗渊.md)。

## 玩家可见

- 新文明在此重建；诸王国与诸城邦林立分布，彼此相对对立。环境相对温和，村镇与城市最为常见，城邦格局已然成形。前文明遗迹仅偶见，日常所见以新文明经营痕迹为主。
- 经 [铁门关（IronGate）](./铁门关.md) 与 [荒地](./荒地.md) 相连；关外视为未知与危险。照不到处统称 [暗渊](./暗渊.md)。

## 玩法关联

- **第一章**：自 [玩家起点](./玩家起点.md) 进入，穿越本地区向 [铁门关](./铁门关.md) 进发（**有日照**）。
- **第四章**：向下途经本地区，**无日照**；沿途救援濒危城市。
- [太阳城（骄阳之城）](./太阳城.md) 横向分布于追日路径一带，与本地区有贸易/冲突关系。
