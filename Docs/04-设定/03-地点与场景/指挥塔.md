# 指挥塔

## 摘要

方舟的控制中心，位于暗渊最深处。终局目的地，也是城主（舰长）完成未竟使命的地方。在这里，可以为太阳提速，改变方舟的命运。

## 基本信息

| 属性 | 内容 |
|------|------|
| 名称 | 指挥塔（内部代号）/ 「遗迹」（居民传说） |
| 类型 | 控制中枢/方舟舰桥 |
| 位置 | 暗渊最深处 |
| 功能 | 控制方舟航行、调节太阳运行、监控全舰状态 |
| 能源 | 一颗骄阳之心，可容纳三颗 |
| 出现章节 | 第三章（发现位置）、第五章（终局抵达） |

## 玩家可见层

### 传说

- 暗渊深处有一座「古老的遗迹」。
- 传说它是「旧神」或「先祖」的居所。
- 传说中，谁能到达遗迹并唤醒沉睡的巨人，就能改变太阳的方向。
- 没有人知道它的真实面貌，进入暗渊的人从未返回。

### 与玩法的关系

- 第三章：从铁巢废墟发现前往指挥塔的线索。
- 第五章：穿越最后的暗渊，抵达指挥塔，终局揭示与抉择。

## 设定母本（内部）

### 真相

- 指挥塔是**方舟的控制中心/舰桥**。
- 位置：暗渊最深处，远离太阳照射的区域。
- 功能：控制方舟航行、调节太阳运行、监控全舰状态。
- 能源：内置一颗 **[骄阳之心](../05-隐秘真相/骄阳之心.md)**，设计上可容纳**三颗**。

### 历史

- **天体撞击事件**：人造太阳轨道被击穿，启动紧急急停。
- **舰长的紧急措施**：
  1. 修改太阳行动模式为「当前模式」（停转但维持光照）
  2. 指挥全方舟居民进入冷冻仓避难
  3. 带少数人驻扎指挥塔，指挥船体修复
- **漫长岁月**：修复工作陷入停滞，指挥塔沉寂于暗渊。
- **终局**：城主（舰长）返回指挥塔，为太阳提速，完成未竟使命。

### 三颗骄阳之心

| 位置 | 状态 | 说明 |
|------|------|------|
| **指挥塔** | 现存，待充能 | 方舟控制中心的核心，终局关键 |
| **太阳城（骄阳之城）** | 现存，被掌控 | 第二颗骄阳之心，第五章关联 |
| **第三颗** | 空缺/位置待定 | 可能位于皮罗斯科罗斯，或需从别处取得 |
