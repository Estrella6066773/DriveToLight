# 设定文档写作约定

← [设定目录](./README.md)

本约定用于减少设定库内的重复与层级混乱。**新增或修订设定词条时须遵守。**

## 信息归属（单一来源）

| 内容类型 | 写在哪里 | 其它文档怎么做 |
|----------|----------|----------------|
| 不向玩家公开的母本真相 | [05-隐秘真相/](./05-隐秘真相/) | 地点/势力词条**只链不写**，禁止复制大段母本 |
| 五章结构与揭示节奏 | [章节划分与故事大纲](./05-隐秘真相/章节划分与故事大纲.md) | 地点词条**不写**完整章节表；最多一句「出现章节」 |
| 四颗骄阳之心分布 | [骄阳之心](./05-隐秘真相/骄阳之心.md)、[指挥塔的真相](./05-隐秘真相/指挥塔的真相.md) | 地点词条只写「与本地点相关的一颗」并链到母本 |
| 玩家可见描述与玩法关联 | 对应地点/势力/世界观词条 | 不写 L1–L4 揭示表 |
| 玩法规则与数值意图 | [02-系统设计/](../02-系统设计/) | 设定词条只写叙事侧，规则链到系统设计 |

## 词条结构（精简）

**地点 / 势力 / 角色**（默认）：

1. **摘要**（一两句）
2. **玩家可见**（居民怎么说、玩家看到什么）
3. **玩法关联**（与哪些系统/章节有关，用链接）
4. **关键关系**（小表，链到相关词条）
5. **待确认**（可选，仅列未决项）
6. **修订记录**

**禁止**在地点词条中增设：`设定母本（内部）`、`揭示层级` 整表、`章节关联` 整表（母本与章节已在专属文档中）。

**05-隐秘真相** 词条：摘要 + 母本正文 + 关键关系 + 待确认；**不写**与 [章节划分与故事大纲](./05-隐秘真相/章节划分与故事大纲.md) 重复的章节表（可一句链过去）。

## 跨词条：只链不抄

- 专名定义、物理本质、层级归属等**只写在归属词条**的正文。
- 其它词条在摘要、关键关系、README 索引中：**只写与本词条相关的一句**，或**仅留链接**；禁止复制已在它处写明的句子（例如：不在 [铁壳](./02-角色与势力/铁壳.md) 写「铁巢废墟是巨大构造体」——见 [铁巢废墟](./03-地点与场景/铁巢废墟.md)）。
- README 索引：一句话定位即可，**不复制**正文定义。

## README 只做索引

- [04-设定/README.md](./README.md) 与各子目录 README：**只列目录与链接**，不复制子文档中的表格正文。
- 详细总览写在子目录 README（如 [03-地点与场景/README.md](./03-地点与场景/README.md)）。
