# 世界概述

## 一句话世界观

自世界诞生以来，太阳便悬于苍穹之巅，永恒不动。它划分了亘古的秩序——「白昼」与「[暗渊](../03-地点与场景/暗渊.md)」各行其道，光与暗皆为永恒。文明在废墟上重生，人们在白昼中繁衍生息，视暗渊为死亡的禁区，从未想过这道界限会移动分毫。然而故事的开端，便是这道亘古秩序的崩解：太阳动了。它不再驻足，而是向着固定的方向缓缓离去，将身后的白昼逐一抛入黑暗——世界正在被暗渊逐步吞没。幸存者的唯一选择，便是驾城追逐那道正在消逝的光，在被吞没之前逃出生天。

## 核心法则

- **太阳**：自先民有记忆以来，太阳便悬于苍穹之巅，永恒不动。它划分了亘古的秩序——「白昼」沐浴永恒之光，「暗渊」沉沦永恒之暗。这道界限是世界的基石：光者恒光，暗者恒暗，从未有逾越之日。人们生于白昼，死于白昼，从未想过这道界限会移动分毫，因为太阳从未动摇。然而故事的开端，便是这道亘古秩序的崩解——太阳正在离去。它向着固定的方向缓缓移动，将身后的白昼逐一抛入黑暗。光熄之处，白昼沦为新的暗渊；暗渊如饥似渴，正在逐步吞没这个世界。这是比末日更古老的恐惧，是比死亡更深邃的绝望——不是末日已过，而是末日正在发生。
- **移动城市**：即循烬城；文明幸存者的载体，由多个可分离、重组的城区组成，可整体移动；停下后方可与外界充分交互。移动规则见 [地图与移动](../../02-系统设计/01-核心系统/地图与移动.md)。
- **荒野**：移动城市之外为正六边形网格上的[荒野地点](../../02-系统设计/02-资源循环/荒野地点.md)，刷新矿区、果地、废墟、村镇、[外部城市](../../02-系统设计/01-核心系统/势力系统.md)等地点，供探索与补给。
- **前进即意义**：停滞意味着被黑暗与严寒吞没；领袖的抉择塑造城市历史。

详细叙事与悬念见 [核心世界观](./核心世界观.md)。地图地区排列见 [地点与场景 · 地图地理关系](../03-地点与场景/README.md#地图地理关系)。
