## 世界是什么样的

### 光与暗

- **太阳**：自文明诞生之初便悬于苍穹、长期**固定不动**；划分「白昼」与「[暗渊](../03-地点与场景/暗渊.md)」。光者恒光，暗者恒暗——这是世人习以为常的秩序。
- **当前时代**：太阳**沿固定方向**（向关外）移离；身后白昼逐片失去照射，沦为新的暗渊。世人面临的是**正在被吞没**的环境，而非已结束的末日。
- **暗渊**：**一切不被太阳照射的区域**；寒冷、危险。民众认知里 **没有「黑夜」** 这一概念，只有暗渊、黄昏、长暗季。
- **日照与地名**：地图上的地区名（如 [日生之地](../03-地点与场景/日生之地.md)）**不表示**当时一定有阳光；白昼范围以外皆为暗渊，而白昼由太阳的位置决定。见 [地图地理关系](../03-地点与场景/README.md#地图地理关系)。

### 废土上的新文明

旧文明早已覆灭（细节见 [世界的真相](../05-隐秘真相/世界的真相.md)）。数代人之后，**日生文明**在废土上重建：

- **聚居形态**：[日生之地](../03-地点与场景/日生之地.md) 为主要活动地带；**诸王国、诸城邦**（专名待定）皆分布于日生之地，彼此相对对立，村镇与城市**最为常见**。
- **与日照**：城邦与村镇定居于日照可达的地带；太阳移离后，失照区段即属 [暗渊](../03-地点与场景/暗渊.md)，聚居区亦会经历长暗季（见 [地图地理关系](../03-地点与场景/README.md#地图地理关系)）。
- **日常景观**：**新文明的经营痕迹占绝对多数**，城邦格局已成形（视觉气质见 [§世界风格与基调](#世界风格与基调)）。
- **前文明遗产**：前文明残骸与遗迹仅在荒野**偶尔**可见，并不随处可见；关外 [荒地](../03-地点与场景/荒地.md) 遗迹相对更多，但仍以荒芜的恶地为主。
- **信仰**：[无敌骄阳会](../02-角色与势力/无敌骄阳会.md) 正典 **[《予日之诗》](../03-关卡与叙事/01-文献/予日之诗.md)** 对日照与迁徙叙事影响最广；其它传说与地方祭仪并存。

## 民众所知


| 概念           | 民众怎么说                                             |
| ------------ | ------------------------------------------------- |
| 太阳           | 信仰与生存依托；曾永恒不动，如今正在离去                              |
| 暗渊           | 一切无日照之处；寒冷而死寂                                     |
| 黑夜           | **不存在**；只有暗渊、黄昏、长暗季                               |
| 前文明          | 存在于传说与荒野中的废墟、巨构                                   |
| 渊光 / 渊光城     | 暗渊深处的传说之地                                         |
| 铁壳           | 关外机械体；与骄阳会敌对                                      |
| 循烬城 / 皮罗斯科罗斯 | 世上唯一的移动巨城；循烬城为中文名，皮罗斯科罗斯（Pyroskóros）为其音译，意为「逐火之蛾」 |


神话与口耳相传语境用 **世人**；日常叙事用 **民众**（见 [设定文档写作约定](../00-设定文档写作约定.md)）。

## 地理与势力

**地理**：纵向主线自下而上为渊光 → … → 铁门关 → [荒地](../03-地点与场景/荒地.md)（关外）→ … → 铁巢；[日生之地](../03-地点与场景/日生之地.md) 为聚居区，诸王国、诸城邦皆在其内，经 [铁门关](../03-地点与场景/铁门关.md) 与关外相连。[太阳城](../03-地点与场景/太阳城.md) 位于太阳正下方，**主线外**横向分布。完整十段顺序与邻接见 [地图地理关系](../03-地点与场景/README.md#地图地理关系)。

**势力**：立场一览见 [世界概述 · 主要势力](./世界概述.md#主要势力)。格局要点——无敌骄阳会以太阳城为首、统摄日生之地武装与宗教秩序，关外由 [猎壳人](../02-角色与势力/猎壳人.md) 狩猎 [铁壳](../02-角色与势力/铁壳.md)；诸城邦与诸王国相对对立，共处日生之地；铁壳据守 [铁巢](../03-地点与场景/铁巢.md)，与无敌骄阳会是死敌。专题见 [角色与势力](../02-角色与势力/README.md)。

## 世界风格与基调

### 类型

本作属于**太阳朋克**范畴下的**后启示录**：艳阳、光能、巨构城市与金色天际线构成视觉主轴；文明断裂、资源紧缺与流亡求存构成戏剧主轴。

### 看见什么

- **新文明为主场**：城邦、村镇与日常经营痕迹占绝大多数画面；社会已在废土上成形，人们在其中生活、贸易与争执。
- **旧世偶发点缀**：前文明遗迹与巨构仅偶尔出现在荒野；关外 [荒地](../03-地点与场景/荒地.md) 相对更多，行进仍以荒芜与险境为主。
- **光与暗对照**：日照区明亮温暖，带着太阳朋克的工业光泽；失照区堕入 [暗渊](../03-地点与场景/暗渊.md)，寒冷而死寂。

### 感受什么

- **持续的压力**：太阳正在离去，停滞即被黑暗吞没；资源永不离线，每前进一步都有代价。
- **短暂的希望**：点亮未知、接纳城镇、探得补给——重压之间仍有喘息。
- **沉重的取舍**：领袖抉择会牺牲城区、关系或人口；希望从不免费。

玩法侧的情绪节奏见 [核心幻想](../../02-系统设计/01-核心体验/核心幻想.md)。
