# 世界的真相

## 摘要

> **引用原文（保留）**
>
> "世界其实是一个巨大的环状方舟，方舟有着极大自转，以至于在方舟内与在地面上无异，天上的人造太阳也会落下真实的阳光，并模拟昼夜与四季，整艘方舟宛如一个大大的摇篮，载着小小的人类驶向远方。但现在方舟早已瘫痪，其中的文明也覆灭许久，就连天上的太阳都陷入异常，不再移动。所幸，幸存的人们在方舟内重新建立起了新的文明，他们在时明时暗、却始终高悬于天空的太阳下诞生，也在太阳的照耀下死去，而不知黑夜为何物。于是当太阳再次开始移动时，谁也没有做好准备。"

世界并非自然星球，而是**环状方舟**。方舟曾载着人类文明航行，直到天体撞击导致瘫痪。原文明覆灭后，新文明在废墟上重生——他们生来就在移动的人造太阳下，不知黑夜，不知方舟，不知自己身在摇篮之中。

本条目为设定母本，**不**向玩家公开。

## 正文

- 后启示录废土世界，文明覆灭又重生。
- 太阳移动，幸存者驾驶城市追逐以延续文明。
- 「暗渊」是太阳照不到的危险区域。

### 设定母本（内部）

#### 一、方舟的结构（世界架构层）

| 属性 | 说明 |
|------|------|
| 形态 | 巨型环状空间站/方舟 |
| 重力 | 自转产生人工重力，内部环境类似行星表面 |
| 世人感知 | 世人完全不知自己身在方舟内，以为这就是「世界」 |
| 当前状态 | **已瘫痪**，不再执行原定航行计划 |

#### 二、文明覆灭的历史（大崩溃）

> **引用原文（保留核心事件）**
>
> "一颗天体击穿了人造太阳的运行轨道，启动紧急功能急停在了后来太阳所在的位置，太阳的停转导致整个方舟陷入异常，并且太阳的异常紧急停止，对方舟的地表，特别是玩家这片区域造成了极其严重的影响，直接变成了一片废墟。"

| 事件 | 说明 |
|------|------|
| 触发 | 天体撞击人造太阳轨道 |
| 紧急措施 | 人造太阳启动急停，固定在撞击点位置 |
| 直接后果 | 方舟陷入异常；撞击区域地表严重损毁，成为废墟 |
| 文明状态 | 原文明覆灭 |
| 时间跨度 | 距游戏开始已有**数代人之久**（具体年数待定） |

#### 三、日生文明的历史（新文明的诞生）

> **引用原文（保留）**
>
> "幸存的人们在方舟内重新建立起了新的文明，他们在时明时暗、却始终高悬于天空的太阳下诞生，也在太阳的照耀下死去，而不知黑夜为何物。"

| 特征 | 说明 |
|------|------|
| 起源 | 废墟上的幸存者后裔 |
| 环境 | 太阳始终高悬，时明时暗（他们以为是正常的昼夜） |
| 认知空白 | **不知黑夜为何物**，没有「夜晚」概念 |
| 语言 | 称太阳照不到的地方为「暗渊」，而非「黑夜」 |
| 技术 | 部分继承前文明遗物（如移动城市），但不知原理 |
| 社会 | 形成了以「追逐阳光」为核心的生存文化 |

#### 四、关键认知差异

| 概念 | 新文明（玩家/世人）理解 | 母本层真相 |
|------|------------------------|-----------|
| 世界 | 一片大陆/废土 | 环状方舟的内表面 |
| 太阳 | 天体，有时会移动 | 人造能源核心，曾停转 |
| 昼夜 | 太阳变暗就是夜晚 | 太阳从未熄灭，只是调光 |
| 暗渊 | 太阳照不到的恐怖区域 | 远离能源核心的黑暗区域 |
| 移动城市 | 追逐太阳的工具 | 前文明遗留的模块化载具 |
