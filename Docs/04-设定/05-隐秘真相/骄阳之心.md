# 骄阳之心

## 摘要

> **引用原文（保留核心设定）**
>
> "驱动太阳城与循烬城运作的器物被称为「骄阳之心」。城主称其为「反应堆」。"
>
> "实际上移动城市使用的燃料相对城市需要的能量只是凤毛麟角，这也解释了为什么即使没有燃料，城市也能在花费相对平时更长的时间拔营后正常行驶，且按照拔营而不按照一段距离消耗燃料，且可以在空城区建造「燃料生产设施」。"
>
> "第一章：骄阳之子点明移动城市使用的骄阳之心，并对循烬城放行。"（新增剧情）

**骄阳之心（城主称为「反应堆」）**是方舟的能源核心，驱动着人造太阳、移动城市，以及方舟的关键系统。它是前文明最高技术的结晶，也是城主（舰长）熟知却对新文明居民保密的秘密。

本条目为设定母本，**不**向玩家公开。

---

## 正文

### 玩家可见层（对外口径）

- 移动城市需要「燃料」才能运转和移动。
- 可以在城区建造「燃料生产设施」来获取更多燃料。
- 某些古老遗迹（如太阳城）似乎拥有「无限的能源」。
- **第一章事件**：[骄阳之子](../02-角色与势力/无敌骄阳会.md)在与城主对峙时**直接点明**循烬城拥有「骄阳之心」，令城主震惊（骄阳之子如何知晓？）
- 城主偶尔会说出「反应堆」这个词，居民以为是某种方言或咒语。

### 设定母本（内部）

#### 一、骄阳之心的本质

| 属性 | 说明 |
|------|------|
| 名称 | **骄阳之心**（新文明居民不知何时开始这样称呼） |
| 舰长术语 | **反应堆**（城主使用的专业术语） |
| 本质 | 前文明的高级能源核心（聚变？零点能？其他？待定） |
| 功能 | 为方舟系统、人造太阳、移动城市提供能源 |
| 特征 | 能量输出巨大，远超新文明居民理解的「燃料」概念 |

#### 二、能量输出与「燃料」的真相

> **引用原文（保留）**
>
> "实际上移动城市使用的燃料相对城市需要的能量只是凤毛麟角"

| 现象 | 居民理解 | 母本层解释 |
|------|----------|-----------|
| 城市需要燃料 | 燃料是能源 | 燃料只是**启动/维持反应堆运行的催化剂/冷却剂**，不是主要能源 |
| 没燃料也能动（慢） | 奇怪的现象/城主的魔法 | 反应堆有**残余能量**可以缓慢释放，无需燃料辅助 |
| 拔营消耗燃料 | 移动需要消耗燃料 | 拔营是**激活反应堆的过程**，消耗的是启动能量，不是行驶能量 |
| 燃料生产设施 | 在生产燃料 | 实际上是在**维护/充能反应堆**或生产催化剂 |
| 太阳城有无限能源 | 神迹/旧神恩赐 | 太阳城拥有**第二颗骄阳之心**，能源几乎无限 |

#### 三、三颗骄阳之心的分布

详见 [指挥塔的真相](./指挥塔的真相.md)。

| 位置 | 状态 | 作用 |
|------|------|------|
| **指挥塔** | 现存，待充能 | 方舟控制中心的核心 |
| **太阳城（骄阳之城）** | 现存，被掌控 | 太阳正下方的城市，拥有第二颗骄阳之心 |
| **第三颗** | 位于**皮罗斯科罗斯（循烬城）** | 第一章被骄阳之子点明，令城主震惊 |

#### 四、城主的知识

- 作为舰长，城主完全理解骄阳之心（反应堆）的工作原理。
- 他使用「反应堆」这个术语，但居民听不懂，将其视为「城主的古老方言」。
- 城主知道：所谓「燃料」只是表象，真正的能源来自骄阳之心。
- 这也是为什么城主能够「修复」或「优化」城市——他实际上是在调整反应堆的输出。

## 与玩法的关系

| 玩法系统 | 对外解释 | 母本层解释 |
|----------|----------|-----------|
| 燃料消耗 | 移动和运转需要消耗资源 | 燃料是催化剂，反应堆才是能源 |
| 无燃料缓慢移动 | 城市勉强运转 | 反应堆残余能量缓慢释放 |
| 燃料生产设施 | 城区建筑产出燃料 | 实际上是在维护反应堆或生产催化剂 |
| 拔营消耗 | 整城移动需要大量准备 | 激活反应堆进入移动模式 |
| 抵达太阳城/指挥塔 | 获得资源/真相 | 接触其他骄阳之心，获得巨大能源或控制权 |
