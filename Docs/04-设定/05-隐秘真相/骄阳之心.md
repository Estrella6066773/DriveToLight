# 骄阳之心

## 摘要

> **引用原文（保留核心设定）**
>
> "驱动太阳城与循烬城运作的器物被称为「骄阳之心」。城主称其为「反应堆」。"
>
> "实际上移动城市使用的燃料相对城市需要的能量只是凤毛麟角，这也解释了为什么即使没有燃料，城市也能在花费相对平时更长的时间拔营后正常行驶，且按照拔营而不按照一段距离消耗燃料，且可以在空城区建造「燃料生产设施」。"
>
> "第一章：骄阳之子点明移动城市使用的骄阳之心，并对循烬城放行。"（新增剧情）

**骄阳之心**（城主称为「反应堆」）是方舟正在使用的一种能源核心，可以驱动指挥塔控制人造太阳。它是前文明最高技术的结晶，也是城主（舰长）熟知却对新文明众人保密的秘密。

本条目为设定母本，**不**向玩家公开。

## 正文

### 玩家可见层（对外口径）

- 移动城市需要「燃料」才能运转和移动。
- 可以在城区建造「燃料生产设施」来获取更多燃料。
- 某些古老遗迹（如太阳城）似乎拥有「无限的能源」。
- **第一章事件**：[骄阳之子](../02-角色与势力/无敌骄阳会.md)在与城主对峙时**直接点明**循烬城拥有「骄阳之心」——骄阳之子通过观察城市运转推断（他们不具备检测技术）。太阳城那颗骄阳之心的最初来源仍未知。

#### 二、能量输出与「燃料」的真相

> **引用原文（保留）**
>
> "实际上移动城市使用的燃料相对城市需要的能量只是凤毛麟角"


| 现象        | 世人理解        | 母本层解释                                          |
| --------- | ----------- | ---------------------------------------------- |
| 城市需要燃料    | 燃料是能源       | 燃料只是**启动/维持反应堆运行的催化剂/冷却剂**，不是主要能源              |
| 没燃料也能动（慢） | 奇怪的现象/城主的魔法 | 反应堆有**残余能量**可以缓慢释放，无需燃料辅助                      |
| 拔营消耗燃料    | 移动需要消耗燃料    | 拔营是**激活反应堆的过程**，消耗的是启动能量，不是行驶能量                |
| 燃料生产设施    | 在生产燃料       | 实际上是在**维护/充能反应堆**或生产催化剂                        |
| 太阳城有无限能源  | 神迹/旧神恩赐     | [太阳城](../03-地点与场景/太阳城.md) 拥有**第三颗骄阳之心**，能源几乎无限 |

#### 四、游戏机制：核心区与移动城区

> **引用设定（新增）**
>
> "反应堆在游戏机制上体现为「核心区」，与核心区相连接的城区为移动城区。"
>
> "移动城市不唯一。如果一个城市被分成两半，只要与任意核心区链接的，都是移动城市的一部分，只有不与核心区连接的，才被视为分离的城区。"

**核心区（反应堆/骄阳之心）**

- 每颗骄阳之心对应一个**核心区**。
- 核心区是移动城市的能源心脏，决定城市能否运转和移动。
- 城主（玩家）可直接与核心区互动，进行维护、优化或调整。

**移动城区**

- 与核心区**物理连接**的城区被视为**移动城区**。
- 移动城区**不坐落在世界地图上**，而是记录在移动城市内部结构中。
- 只有当城区与核心区**断开连接**时，才被视为**分离的独立城区**，落入世界地图网格。

**城市分割机制**

- 一个城市可以被**分成两半**（例如被地形、敌人攻击、或玩家主动分割）。
- 只要城区仍与**任意核心区**保持连接，它就属于**移动城市的一部分**。
- 这允许玩家构建**多核心城市**（若拥有多颗骄阳之心），或进行**模块化城市设计**。

#### 五、城主的知识

- 作为舰长，城主完全理解骄阳之心（反应堆）的工作原理。
- 他使用「反应堆」这个术语，但旁人听不懂，将其视为「城主的古老方言」。
- 城主知道：所谓「燃料」只是表象，真正的能源来自骄阳之心。
- 这也是为什么城主能够「修复」或「优化」城市——他实际上是在调整反应堆的输出。
- 城主知晓核心区机制，能够规划城市的物理连接结构。

## 与玩法的关系


| 玩法系统      | 对外解释        | 母本层解释               |
| --------- | ----------- | ------------------- |
| 燃料消耗      | 移动和运转需要消耗资源 | 燃料是催化剂，反应堆才是能源      |
| 无燃料缓慢移动   | 城市勉强运转      | 反应堆残余能量缓慢释放         |
| 燃料生产设施    | 城区建筑产出燃料    | 实际上是在维护反应堆或生产催化剂    |
| 拔营消耗      | 整城移动需要大量准备  | 激活反应堆进入移动模式         |
| 抵达太阳城/指挥塔 | 获得资源/真相     | 接触其他骄阳之心，获得巨大能源或控制权 |
