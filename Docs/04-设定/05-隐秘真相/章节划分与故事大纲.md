## 第一章：初速度

太阳渐远。循烬城赶在黄昏离去、暗渊降临之前完成建造，开拔启程。城市向着太阳的方向前进，追逐白昼，驶向日生之地。

循烬城抵达日生之地，与当地城市、村镇接触。此地诸王国土与城邦同盟林立分布，组成数个大小不一的势力，彼此相对对立。

随着循烬城前进，有关它的传闻在日生之地传播开来。伴随传闻一同散开的，还有太阳开始移动的异象流言——这使得循烬城逐渐成为比渊光城更加知名的传说。

在日生之地的诸多势力中，最为强大的是无敌骄阳会。循烬城的名声远扬，连骄阳之子也有所耳闻。但这对循烬城而言并非好事——很快，城市便驶入了无敌骄阳会的地界，而这里新建的防御工事与四处游荡的军队，对循烬城并不友善。

不过循烬城对此早有准备。依靠提前接纳的佣兵之城**安德雷亚Andreia**与工匠之城**赫菲斯提亚Hephaistia**，循烬城得以突破无敌骄阳会的封锁，越过其首都太阳城，前往日生之地的边境：铁门关。

在循烬城将太阳城甩在身后、前往铁门关的途中，骄阳之子发来通讯。通讯中，骄阳之子坦白了自己对骄阳之心即反应堆的了解，解释了下令阻拦循烬城的理由：夺取骄阳之心，将其作为能源，使太阳城能够在未来迁入暗渊后生存下去。

骄阳之子告知城主，自己已放弃袭击循烬城，并命令铁门关守军放行。同时，他祈求城主解决铁巢的问题：太阳正移向铁巢方向，那里还有一颗被铁壳霸占的骄阳之心。历史上，形如鸟巢的巨大铁壳——巢主，曾从日生之地夺走过一颗骄阳之心，逃入暗渊后苟活至今；如今仍在铁门关外出没的铁壳集群，便是巢主仍存活的证明。

### 地理路径

**向上 · 有日照**：[玩家起点](../03-地点与场景/玩家起点.md) → [日生之地](../03-地点与场景/日生之地.md) → [太阳城](../03-地点与场景/太阳城.md) → [铁门关](../03-地点与场景/铁门关.md)

### 关键剧情细节

**骄阳之子的通讯**（前往铁门关的途中）：

骄阳之子坦白自己知晓骄阳之心即反应堆，解释阻拦理由：夺取骄阳之心为太阳城供能，使其能在未来迁入暗渊后生存。他告知已放弃袭击并命令铁门关守军放行，同时**祈求城主解决铁巢问题**——太阳正移向铁巢，那里有一颗被铁壳霸占的骄阳之心。

**铁巢历史**：形如鸟巢的巨大铁壳——巢主，曾从太阳城夺走一颗骄阳之心，逃入暗渊后苟活至今。铁门关外游荡的铁壳集群，便是巢主仍存活的证明。

## 第二章：角速度

穿越铁门关，经荒地、暗渊(荒地方向)，抵达铁巢获取第二颗骄阳之心。

### 地理路径

**向上 · 有日照**：[铁门关](../03-地点与场景/铁门关.md) → [荒地](../03-地点与场景/荒地.md) → [暗渊](../03-地点与场景/暗渊.md)（荒地方向）→ [铁巢废墟](../03-地点与场景/铁巢废墟.md) → [铁巢](../03-地点与场景/铁巢.md)

### 关键流程

1. **进入荒地**：遭遇猎壳人与铁壳，选择站队
2. **太阳底下跋涉**：继续向上，穿越暗渊(荒地方向)——追日途中有日照
3. **速度差拉大**：太阳加速，城市速度有限，差距越拉越大
4. **穿越铁巢废墟**：发现方舟结构遗迹
5. **铁巢终局**：与巢主对峙，依关系获取第二颗骄阳之心
6. **章末转向**：确认追日策略失效，决定转向暗渊



### 站队分支


| 路径    | 做法   | 收益              | 代价      |
| ----- | ---- | --------------- | ------- |
| 协助猎壳人 | 猎杀铁壳 | 金属、能源可观；猎壳人营地开放 | 铁壳会反击   |
| 与铁壳交好 | 接受物资 | 单次少但铁壳到处都有      | 骄阳会视为亵渎 |




### 揭示内容


| 层级    | 内容           | 触发             |
| ----- | ------------ | -------------- |
| L1    | 世界是环状方舟      | 铁巢废墟遗迹         |
| L2    | 太阳是人造核心，正在加速 | 废墟记录/城主解释      |
| L1-L2 | 第二颗骄阳之心存在    | 铁巢中枢，巢主掌控/攻城夺取 |

## 第三章：离心力

不再追逐太阳，反向进入暗渊，向渊光城进发。

### 地理路径

**向下 · 无日照**：[铁巢](../03-地点与场景/铁巢.md) → [铁巢废墟](../03-地点与场景/铁巢废墟.md) → [暗渊](../03-地点与场景/暗渊.md)（荒地方向）→ [荒地](../03-地点与场景/荒地.md) → [铁门关](../03-地点与场景/铁门关.md) / [日生之地](../03-地点与场景/日生之地.md)

### 核心叙事

**反转**：背离太阳，向暗渊冲锋。

太阳最后余晖消失，循烬城在无日照的暗渊中移动。地理上经过荒地、铁门关、日生之地，但环境上均为暗渊。

> **玩法对齐**（系统设计）：太阳照射区沿**纵向**移动；当照射区**完全移出**有界地图、**全图均为暗渊带**时 **`sun_motion_enabled=false`**（叙事上对齐第三章「余晖消失」）。见 [地图与移动 · 太阳照射与移动停用](../../02-系统设计/02-地图与世界/地图与移动.md#太阳照射区与移动停用)。

城主知道：暗渊最深处有指挥塔，有改变一切的钥匙。

## 循烬城名称

**循烬城**（Pyroskóros / **皮罗斯科罗斯**）：中文名**循烬城**；**皮罗斯科罗斯**为其音译，希腊语意为「逐火之蛾」。二者指**同一座**移动城市——世上唯一可整城迁徙的前文明载具。寓意：城市追逐骄阳，却不知所追之「太阳」是人造余烬，正如它自己也是前文明遗留的余烬。剧情推进后接入、随城迁徙的城区，亦属循烬城。
