## 第一章：初速度

> **引用原文（保留）**
>
> "第一章：向心力，从主城在黄昏离去前完工开拔，追赶太阳开始，到突破太阳城的边境，前往荒地结束。"

### 核心叙事

主城**皮罗斯科罗斯（循烬城，Pyroskóros）**在「黄昏」离去前完工开拔。「黄昏」是居民对太阳变暗时段的称呼——他们不知道这是太阳停转后的异常，以为只是正常的「傍晚」。

城主（玩家）知道真相：太阳正在离去，如果不追赶，城市将陷入暗渊。

### 玩法循环

- **追逐太阳**：在太阳完全离开当前区域前，建立基础设施并开拔
- **资源积累**：收集建材、食物、燃料、人口
- **首次远征**：突破太阳城边境，进入未知荒地

### 地理节点

1. **起始点**：[玩家起点](../03-地点与场景/玩家起点.md)（白昼与暗渊交界，建立主城）
2. **穿越日生之地**：向上，经 [日生之地](../03-地点与场景/日生之地.md) 向太阳方向进发
3. **[太阳城（骄阳之城）](../03-地点与场景/太阳城.md)边境**：与[无敌骄阳会](../02-角色与势力/无敌骄阳会.md)的冲突与封锁（横向分布于追日路径）
4. **[铁门关（IronGate）](../03-地点与场景/铁门关.md)**：第一章终点，前往荒地的最后门户

### 揭示内容


| 层级        | 内容                    | 对玩家/角色的意义                                                 |
| --------- | --------------------- | --------------------------------------------------------- |
| L0        | 必须追逐太阳，否则环境恶化         | 建立游戏核心驱动力                                                 |
| L0 暗示     | 城主偶尔会说出「反应堆」「轨道」等古怪词汇 | 埋下身份伏笔（居民听不懂）                                             |
| L3 暴露（可选） | 骄阳之子点明循烬城拥有「骄阳之心」     | 城主可选择是否承认/如何回应                                            |
| L1 线索     | 铁壳被太阳照射后会恢复活力         | 解释为何铁壳不靠近[铁门关（IronGate）](../03-地点与场景/铁门关.md)（铁门关在太阳照射范围内） |


### 关键事件

- 主城完工，首次拔营
- **穿越日生之地**：循烬城越过整个日生之地，向太阳城进发
- **太阳城封锁**：[无敌骄阳会](../02-角色与势力/无敌骄阳会.md)试图阻拦循烬城，意图不明
- **突破封锁**：循烬城凭借骄阳之心的力量突破太阳城军队封锁
- **关键对话**：城主与[骄阳之子](../02-角色与势力/无敌骄阳会.md)对峙
  - 骄阳之子**点明**循烬城拥有「骄阳之心」（反应堆）
  - 骄阳之子承认：本想夺取骄阳之心为太阳城供能，让更多人活下去
  - 但连首都军队都无法拦下循烬城，于是放弃并放行
- **铁门关（IronGate）的防御逻辑**：[铁门关（IronGate）](../03-地点与场景/铁门关.md)军队不能用来阻拦循烬城，因为他们必须防范被太阳照射后恢复活力的[铁壳](../02-角色与势力/铁壳.md)
- 穿越[铁门关（IronGate）](../03-地点与场景/铁门关.md)，完成第一章（第二章由此进入 [荒地](../03-地点与场景/荒地.md)）

## 第二章：角速度

> **引用原文（保留）**
>
> "第二章：角速度，从穿越铁门关，离开日生之地开始，通过荒地，穿越铁巢废墟地区，到抵达深处的铁巢结束。"
>
> "太阳正在加速，而最初的速度不够快，所以从太阳离去到再次回归，人类需要数月（待定）活在黑暗中，这就是为什么玩家的城市要去追逐太阳。而后来，当太阳越来越快，玩家往前走就没有用了。因为城市的速度是有限的，你不可能永远追得上太阳（第二章：角速度）"

### 核心叙事

太阳开始**加速移动**。最初，太阳速度太慢，导致「离去到回归」之间有数月黑暗（居民称之为「长暗季」）。这就是为什么要追逐太阳——为了避开漫长的暗渊。

但太阳越转越快，玩家的城市**追不上了**。城市的速度有限，不可能永远追上太阳。

当主城穿越**铁巢废墟**地区，抵达深处的**铁巢**时，玩家作为城主，做出重大决定：**不再追逐太阳，而是就地收集资源，准备向暗渊冲锋**。

### 玩法循环

- **速度竞赛**：太阳移动速度逐渐加快，追赶窗口变短
- **资源压力**：荒地资源并不贫瘠，但采集过程被高强度战斗风险包围
- **遭遇铁壳**：[铁门关（IronGate）](../03-地点与场景/铁门关.md)之外的荒地中游荡着[铁壳](../02-角色与势力/铁壳.md)，击败可获取建材与燃料
- **关系建立**：与铁壳势力的互动影响最终获取骄阳之心的方式
- **核心目标**：穿越[铁巢废墟](../03-地点与场景/铁巢废墟.md)地区，抵达深处的[铁巢](../03-地点与场景/铁巢.md)，获取**第二颗骄阳之心**
- **决策点**：继续前进（徒劳）还是转向暗渊（未知）？
- **可见势力限定（第二章）**：荒地中仅出现两类对象：
  - [无敌骄阳会](../02-角色与势力/无敌骄阳会.md)阵营的猎壳人队伍与基地（城市/村镇格子）
  - [铁壳](../02-角色与势力/铁壳.md)各类单位

### 地理节点

1. **起点**：穿越[铁门关（IronGate）](../03-地点与场景/铁门关.md)，进入 [荒地](../03-地点与场景/荒地.md)
2. **荒地边缘**：遭遇[铁壳](../02-角色与势力/铁壳.md)
3. **暗渊（太阳经过段）**：太阳过顶而地面无照，向上穿越 [暗渊](../03-地点与场景/暗渊.md) 区段
4. **终点**：[铁巢](../03-地点与场景/铁巢.md)（经 [铁巢废墟](../03-地点与场景/铁巢废墟.md)）

### 揭示内容


| 层级    | 内容              | 触发方式                                    |
| ----- | --------------- | --------------------------------------- |
| L1    | 世界是环状方舟         | [铁巢废墟](../03-地点与场景/铁巢废墟.md) 地区内发现方舟结构遗迹 |
| L1-L2 | **第二颗骄阳之心存在**   | [铁巢](../03-地点与场景/铁巢.md) 中枢，巢主掌控/攻城夺取    |
| L2    | 太阳是人造核心，正在加速    | 废墟中的古老记录/城主的解释                          |
| L2    | 「夜晚」是相对的，太阳从未熄灭 | 城主透露部分真相给副官（可选）                         |


### 关键事件

- 穿越[铁门关（IronGate）](../03-地点与场景/铁门关.md)，进入真正未知的领域
- **遭遇[铁壳](../02-角色与势力/铁壳.md)**：击败游荡的机械体，获取建材与燃料补给
- **关系建立**：与铁壳势力的互动（战斗？回避？交易？）建立关系；规则见 [外部城市与组织关系](../../02-系统设计/01-核心系统/外部城市与组织关系.md)
- 发现[铁壳](../02-角色与势力/铁壳.md)与[无敌骄阳会](../02-角色与势力/无敌骄阳会.md)的敌对关系（可利用？）
- 发现追赶太阳已不可能（速度墙）
- **穿越[铁巢废墟](../03-地点与场景/铁巢废墟.md)地区，抵达 [铁巢**](../03-地点与场景/铁巢.md)：
  - 与**巢主**对峙——如同骄阳之子之于太阳城，巢主代表铁巢与铁壳势力
  - **巢主依赖第二颗骄阳之心存活至今**——失去它意味着「死亡」或休眠
  - **依关系获取第二颗骄阳之心**：
    - **关系友好**：巢主认可玩家，直接赠送骄阳之心（视为值得尊敬的对手或某种「同类」）
    - **关系敌对**：通过**攻城玩法**夺取骄阳之心
- **决定转向**：准备向暗渊冲锋

## 第三章：离心力

> **引用原文（保留）**
>
> "第三章：离心力，主城从离开铁巢开始，到穿过荒地，回到人类生存的土地结束。"
>
> "当太阳最后的余晖也离主城而去的时候，主城终于完成了拔营，并开始向暗渊发起冲锋"

### 核心叙事

**反转**：不再追逐太阳，而是**背离太阳**，向暗渊冲锋。

当太阳最后的余晖离去，主城终于完成准备，开始向暗渊深处进发——**太阳已不在头顶**，此后全程在**无日照**的暗渊中移动。这是居民从未有过的体验；地理上仍会经过荒地、铁门关、日生之地等**命名地区**，但环境上**一律是暗渊**。

但城主知道：暗渊最深处有指挥塔，有改变一切的钥匙。

### 玩法循环

- **逆向生存**：在没有太阳的区域维持城市运转
- **暗渊探索**：环境极端恶劣，资源极其稀缺
- **心理压力**：居民恐慌，需要城主维持士气

### 地理节点

1. **起点**：[铁巢](../03-地点与场景/铁巢.md)/[铁巢废墟](../03-地点与场景/铁巢废墟.md) 边缘，**太阳最后余晖消失**
2. **向下、无日照**：向下，经 [暗渊](../03-地点与场景/暗渊.md)（太阳经过段）→ [荒地](../03-地点与场景/荒地.md) → [铁门关](../03-地点与场景/铁门关.md)
3. **终点**：地理上进入 [日生之地](../03-地点与场景/日生之地.md) 区段（原文「人类生存的土地」）——**仍无日照**，属暗渊

### 揭示内容


| 层级  | 内容        | 触发方式             |
| --- | --------- | ---------------- |
| L3  | 城主是方舟舰长   | 发现指挥塔的准入权限只对城主开放 |
| L3  | 指挥塔可以控制太阳 | 废墟中的舰长日志/指挥塔初步接触 |


### 关键事件

- 太阳余晖消失，进入完全黑暗
- 向暗渊发起冲锋
- 发现指挥塔的位置
- 副官对城主的质疑达到顶峰

## 第四章：摩擦力

> **引用原文（保留）**
>
> "第四章：摩擦力，从越过铁门关，回到日生之地开始，到回到玩家起始点"
>
> "一路上，主城循着地面上城市的行迹，一路穿过铁巢，回到那曾经属于人的土地，那里的城市还在苦苦坚持，但看起来坚持不了多久了，你可以分一些物资给他们，让他们度过寒冬，也可以接纳他们，将他们的城市连接到主城。但是，你或许可以救下一座城和其中的人们，却没法救下每一座城。随着行进的继续，仍然明亮的城市越来越少。城市尚有燃料，或许能活到太阳再次降临，但更多的城市，都不幸地成为了你地图上标记着的废墟。"

### 核心叙事

**太阳已离去**，主城在**无日照**的暗渊中继续向下。地理上自 [铁门关](../03-地点与场景/铁门关.md) 经 [日生之地](../03-地点与场景/日生之地.md) 至 [玩家起点](../03-地点与场景/玩家起点.md)——地区名仍是曾经人类聚居之地，**当时已无阳光**。沿途城市在长暗季中挣扎，仍被太阳照到的城市越来越少。

**道德抉择**：

- 分物资给它们，让它们多活一会儿？
- 接纳它们，将城市连接到主城，增加负担？
- 还是直接离开，保存自己的资源？

**真相**：你救不了一座城市，更救不了每一座城市。这是资源与道德的张力。

### 玩法循环

- **救援抉择**：每座遭遇的城市都是道德考验
- **资源管理**：接纳意味着更多人口、更多消耗
- **路线规划**：选择救哪些城，放弃哪些城

### 地理节点

1. **起点**：越过 [铁门关（IronGate）](../03-地点与场景/铁门关.md)（第三章末；**无日照**）
2. **沿途**：向下穿越 [日生之地](../03-地点与场景/日生之地.md) 区段，救援濒危城市（含 [太阳城（骄阳之城）](../03-地点与场景/太阳城.md) 一带冲突）
3. **终点**：[玩家起点](../03-地点与场景/玩家起点.md)（**无日照**）

### 揭示内容


| 层级  | 内容                            | 触发方式            |
| --- | ----------------------------- | --------------- |
| L3  | 城主身份被同伴知晓/部分公开（可选向最亲近的副官完全坦白） | 道德抉择：是否坦白、向谁坦白  |
| L3  | 骄阳之心的真相                       | 发现某些城市拥有/曾拥有反应堆 |


### 关键事件

- 遭遇第一座濒危城市，首次救援抉择
- 遭遇[太阳城（骄阳之城）](../03-地点与场景/太阳城.md)，与[无敌骄阳会](../02-角色与势力/无敌骄阳会.md)的第二次冲突
- **突破铁门关**：与[无敌骄阳会](../02-角色与势力/无敌骄阳会.md)发生冲突，最终赢得尊重与放行
- 更多城市变为废墟
- 最终决定前往指挥塔

## 第五章：向心力

> **引用原文（保留）**
>
> "第五章：没想好，总之就是揭示反应堆设定，然后主城前往以前一直在暗渊深处的「指挥塔」，用反应堆给指挥塔功能，并在指挥塔调整太阳的行动策略，为太阳提速，此时揭示，指挥塔也有一个骄阳之心，并且设计上能容纳三颗骄阳之心，还有一颗在太阳城。"
>
> "玩家是方舟的舰长，按照紧急处理流程，将人造太阳的行动模式修改成当前的模式，并指挥全方舟的人进入舰体内部的冷冻仓避难，自己则带着少数人驻扎在指挥塔，指挥船体的修复。"

### 核心叙事

终局：**前往指挥塔，为太阳提速**。

- 用反应堆（骄阳之心）为指挥塔供能
- 在指挥塔调整太阳的行动策略
- 揭示指挥塔可容纳三颗骄阳之心（已有两颗：指挥塔一颗、太阳城一颗）

## 主城名称释义

> **引用原文（保留）**
>
> "玩家主城的名字：Pyroskóros（皮罗斯科罗斯，循烬城），意为逐火之蛾。"

**Pyroskóros**（希腊语词根组合）：

- **Pyr** = 火（pyro-）
- **Skóros** = 追踪者、追随者（scoros）

中文译名「循烬城」：

- **循** = 追逐、遵循
- **烬** = 火焰的余烬、太阳的光芒
- **城** = 移动城市

**寓意**：如同飞蛾追逐火焰，这座城市追逐着天穹上的骄阳。它不知道的是，它所追逐的「太阳」并非自然之火，而是人造的余烬——正如它自己，也是前文明遗留的「余烬」。
