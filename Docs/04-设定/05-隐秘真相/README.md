## 关键概念速查

### 世界的本质

- **环状方舟**：巨型环形空间站，通过自转产生重力
- **天体撞击**：导致方舟瘫痪、原文明覆灭的事件
- **文明覆灭**：原文明因撞击与太阳停转而毁灭
- **日生文明**：在太阳照耀下重建的新文明，不知黑夜

### 太阳与昼夜

- **人造太阳**：方舟的照明/能源核心，非自然天体
- **模拟昼夜**：太阳调整明暗模拟昼夜，但从未熄灭
- **真正的黑夜**：前文明时代的完全黑暗，新文明从未见过
- **暗渊**：太阳照射不到的区域，新文明对「黑暗」的称呼

### 人物身份

- **城主/玩家**：表面是移动城市领袖，实则是方舟舰长
- **舰长**：前文明时代方舟的最高指挥官，亲历覆灭
- **城主知道黑夜**：舰长来自有真正昼夜的时代

### 技术与能源

- **骄阳之心**：驱动太阳与城市的能源核心（城主称为「反应堆」）
- **三颗分布**：指挥塔一颗、太阳城一颗、第三颗待定
- **燃料的真相**：燃料只是催化剂/冷却剂，骄阳之心才是能源

### 地点

- **指挥塔**：方舟控制中心，位于暗渊深处，可控制太阳
- **太阳城（骄阳之城）**：位于太阳正下方的城市，拥有第二颗骄阳之心
- **铁门关**：日生之地边界，离开安全区的门户
- **铁巢废墟**：接近指挥塔的区域，有重要线索
- **皮罗斯科罗斯（循烬城）**：玩家的移动城市，可能拥有第三颗骄阳之心

### 势力

- **骄阳之子（教主+皇帝）**：太阳城的统治者，控制第二颗骄阳之心
- **无敌骄阳会**：与玩家发生冲突的组织，后期可赢得尊重与放行
