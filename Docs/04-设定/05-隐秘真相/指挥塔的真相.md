# 指挥塔的真相

## 摘要

> **引用原文（保留核心设定）**
>
> "主城前往以前一直在暗渊深处的「指挥塔」，用反应堆给指挥塔供能，并在指挥塔调整太阳的行动策略，为太阳提速。"
> 
> "指挥塔也有一个骄阳之心，并且设计上能容纳三颗骄阳之心，还有一颗在太阳城。"
> 
> "玩家是方舟的舰长，在航行过程中，一颗天体击穿了人造太阳的运行轨道，启动紧急功能急停在了后来太阳所在的位置，太阳的停转导致整个方舟陷入异常，并且太阳的异常紧急停止，对方舟的地表，特别是玩家这片区域造成了极其严重的影响，直接变成了一片废墟。舰长按照紧急处理流程，将人造太阳的行动模式修改成当前的模式，并指挥全方舟的人进入舰体内部的冷冻仓避难，自己则带着少数人驻扎在指挥塔，指挥船体的修复。"

指挥塔是方舟的**控制中心**，位于暗渊深处。它曾是舰长（城主）在文明覆灭时的指挥中心，也是最终可以改变方舟命运的地方。在这里，城主可以为太阳提速、改变航行策略——甚至决定方舟的未来航向。

本条目为设定母本，**不**向玩家公开。指挥塔应在**后期章节**（第四章末至第五章）揭露。

## 正文

### 玩家可见层（对外口径）

- 「指挥塔」是暗渊深处的古老遗迹，据说是「旧神」或「先祖」的居所。
- 传说中，谁能到达指挥塔并唤醒沉睡的巨人，就能改变太阳的方向。
- （新文明居民对其一无所知，只有城主知道它的真正意义。）

### 设定母本（内部）

#### 一、指挥塔的本质

| 属性 | 说明 |
|------|------|
| 真实身份 | 方舟的**控制中心/舰桥** |
| 位置 | 位于**暗渊深处**，远离太阳照射的区域 |
| 功能 | 控制方舟航行、调节太阳运行、监控全舰状态 |
| 能源 | 内置一颗 **[骄阳之心](./骄阳之心.md)**，可接入更多骄阳之心增强功率 |

#### 二、历史：文明覆灭时的指挥中心

> **引用原文（保留）**
>
> "舰长按照紧急处理流程，将人造太阳的行动模式修改成当前的模式，并指挥全方舟的人进入舰体内部的冷冻仓避难，自己则带着少数人驻扎在指挥塔，指挥船体的修复。"

| 时间点 | 事件 |
|--------|------|
| 天体撞击 | 人造太阳轨道被击穿，启动紧急急停 |
| 紧急流程 | 舰长修改太阳行动模式为「当前模式」（停转但维持光照） |
| 避难指令 | 指挥全方舟居民进入冷冻仓 |
| 驻扎指挥塔 | 舰长带少数人留守，指挥修复工作 |
| 漫长岁月 | 修复工作陷入停滞，指挥塔沉寂于暗渊 |

#### 三、骄阳之心的三颗分布

> **引用原文（保留）**
>
> "指挥塔也有一个骄阳之心，并且设计上能容纳三颗骄阳之心，还有一颗在太阳城。"

| 位置 | 骄阳之心状态 | 说明 |
|------|--------------|------|
| **指挥塔** | 现存，待充能 | 方舟控制中心的核心，可为太阳提速 |
| **太阳城（骄阳之城）** | 现存，被教主/皇帝掌控 | 位于太阳正下方的城市，拥有第二颗骄阳之心 |
| **第三颗** | 空缺/位置待定 | 可能位于皮罗斯科罗斯，或需从别处取得 |

#### 四、最终目标：为太阳提速

| 阶段 | 内容 |
|------|------|
| 现状 | 太阳已停转，缓慢移动或固定 |
| 问题 | 太阳速度太慢，无法覆盖整个方舟；暗渊区域持续扩大 |
| 解决方案 | 用骄阳之心为指挥塔供能，调整太阳行动策略，**为太阳提速** |
| 结果 | 太阳恢复足够速度，可以重新照亮整个方舟 |
| 揭示 | 舰长（城主）终于完成未竟的使命 |
