# 指挥塔的真相



## 摘要



**指挥塔**是方舟控制中心/舰桥，位于暗渊深处。口语中称高塔为 **[渊光](../03-地点与场景/渊光.md)**，其脚下传说城市为 **[渊光城](../03-地点与场景/渊光城.md)**。终局：集齐四颗 [骄阳之心](./骄阳之心.md)，为指挥塔供能并为太阳提速。



> 引用设定：「主城前往暗渊深处的指挥塔，用反应堆供能，调整太阳行动策略，为太阳提速。」

## 母本



### 本质与功能



- 位置：暗渊深处；内置**第四颗骄阳之心**，设计上可接入**全部四颗**。

- 功能：控制方舟航行、调节太阳运行、监控全舰状态。



### 文明覆灭时的指挥



天体撞击击穿人造太阳轨道 → 紧急急停 → 舰长修改太阳为「当前模式」→ 众人入冷冻仓 → 舰长带少数人驻扎指挥塔指挥修复 → 修复停滞，指挥塔沉寂。
