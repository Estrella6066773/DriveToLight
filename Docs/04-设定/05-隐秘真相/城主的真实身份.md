# 城主的真实身份

## 摘要

> **引用原文（保留核心设定）**
>
> "玩家是方舟的舰长（剧情最高机密）"

玩家所扮演的「城主」，**一直都知道自己真实的身份**——他是**方舟的舰长**，前文明时代的人物，曾亲历天体撞击、文明覆灭，并按照紧急处理流程指挥了方舟的应急措施。如今他以「城主」的身份带领新文明的人类，但**从未向任何人透露过自己的真实身份**。

本条目为设定母本，**不向新文明居民公开**。城主可以选择**何时、向谁、透露多少真相**，这是一个贯穿全篇的道德张力——不是「觉醒」，而是「是否坦白」。

## 正文

### 玩家可见层（对外口径）

- 玩家是移动城市「皮罗斯科罗斯（Pyroskóros，循烬城）」的城主。
- 城主是文明的领袖，带领幸存者追逐太阳、延续文明。
- 城主似乎比其他人更了解旧世界的技术（如移动城市的运作）。
- 城主偶尔会说出一些新文明居民听不懂的古老词汇（如「反应堆」「舰长」等，居民以为是某种方言或咒语）。

### 设定母本（内部）

#### 真实身份：方舟舰长

| 属性 | 说明 |
|------|------|
| 真实身份 | 方舟的**舰长**，前文明最高指挥官之一 |
| 经历 | 亲历天体撞击事件，执行紧急应对流程 |
| 紧急措施 | 修改太阳行动模式、指挥居民进入冷冻仓、带领少数人驻扎指挥塔 |
| 沉睡/隔离 | 在漫长岁月中处于某种休眠或隔离状态（具体机制待定） |
| 苏醒 | 在新文明时代以「城主」身份重新活动 |

#### 为何城主知道「真正的黑夜」

- 城主来自**前文明时代**，那时太阳正常运转，有真正的昼夜交替。
- 他亲身体验过**真正的夜晚**——太阳完全熄灭、黑暗降临的状态。
- 新文明居民从未见过这种状态，他们以为太阳「变暗」就是夜晚。
- 这是城主与新文明居民的根本认知差异——城主一直知道真相，但选择是否告知他人。

#### 城主的知识与行为（城主自知，他人不知）

| 现象 | 居民理解 | 母本层解释（城主自知） |
|------|----------|----------------------|
| 城主懂旧技术 | 城主聪明/有传承 | 舰长本就掌握方舟系统操作 |
| 城主说古怪词汇 | 城主的方言/咒语 | 舰长在使用前文明术语（如「反应堆」），有时是不小心脱口而出 |
| 城主执着于追逐太阳 | 这是生存本能 | 舰长知道太阳可以被追上、被控制 |
| 城主知道「黑夜」概念 | （居民没有这个概念） | 舰长来自有真正昼夜的时代，但选择是否解释 |
| 城主对指挥塔的熟悉 | （居民从未见过） | 舰长曾在此驻扎指挥修复工作 |

## 叙事节奏：不是觉醒，是暴露与选择

### 早期暗示（第一章）——城主的「口误」

- 城主偶尔会说出居民听不懂的词：「反应堆」「轨道」「舰长」（居民以为是某种头衔或咒语）。
- 城主对移动城市的操作异常熟练，仿佛「本就知道怎么开」。
- 某些旧技术装置只有城主能激活。
- **关键**：玩家（操作者）从一开始就通过城主的言行**感知到他知道更多**，而非突然觉醒。

### 中期张力（第二章-第三章）——质疑与隐瞒

- 发现指挥塔遗迹时，城主表现出异常的熟悉感，同伴开始怀疑。
- 其他角色直接质问：「城主，你究竟知道多少我们不知道的？」
- **玩家选择**：向最亲近的副官透露部分真相？继续隐瞒？半真半假？
- 每次透露都是**不可逆的**——知情者可能泄密，也可能成为盟友。

### 后期暴露（第四章-第五章）——无法再藏

- 到达指挥塔，城主自动获得系统权限，**物理层面无法否认**。
- 同伴震惊的不是「你是舰长？」，而是「你早就知道，却不说？」
- **核心道德抉择**：向所有居民公开真相？继续守护秘密？选择性告知？
- 不同选择影响终局氛围与多重结局走向。
