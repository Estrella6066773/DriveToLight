# 设定

本目录存放《循光之城》的**世界观、角色、地点、时间线**等叙事与背景。

> 对外读者请先从 [游戏介绍](../游戏介绍.md) 入手；本目录各文档为世界观专题详解。

## 子目录

| 目录 | 用途 |
|------|------|
| [01-世界观/](01-世界观/) | 世界起源、核心法则、叙事主题 |
| [02-角色与势力/](02-角色与势力/) | 主要角色、NPC、势力组织 |
| [03-地点与场景/](03-地点与场景/) | 城市分区、关键场景、地图设定 |
| [04-时间线/](04-时间线/) | 历史年表、关键事件 |
| [05-隐秘真相/](05-隐秘真相/) | **内部设定母本**：不向玩家公开的后期悬念（如太阳的真相） |
