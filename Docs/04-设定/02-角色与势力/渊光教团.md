# 渊光教团

## 摘要

前文明覆灭前后，驻守 [指挥塔](../05-隐秘真相/指挥塔的真相.md)（世人称 [渊光](../03-地点与场景/渊光.md)）修复班次的**太阳管控祭司阶层**；城主本人即**初代牧首**。教团在修复停滞后瓦解，遗存化为暗渊深处「渊光城灯火」等传说；当代游戏中**无完整组织**，仅碎片见于口述与遗迹。

## 玩家可见

- 暗渊老人偶言：渊光城有「旧日牧者」的灯火，从未熄灭。
- 与 [渊光城](../03-地点与场景/渊光城.md) 传说混说，无人分得清教团与城市的区别。
- **不向玩家直称**「渊光教团」；宜用「塔下旧日信仰」「暗渊深处的牧者传说」等说法。

## 设定母本（内部）

| 属性 | 说明 |
|------|------|
| 成立时点 | **覆灭前后紧急期**：指挥官率少数人驻扎指挥塔指挥修复时，同步设立塔下驻守者的仪式与记录职能 |
| 与指挥塔关系 | **祭司阶层即太阳管控编制**的宗教封装；牧首 = 指挥官在塔下的仪式头衔，非平行权力中心 |
| 牧首职能 | 主持塔下仪式、记录修复进度、保管圣痕图谱；与太阳子系统操作权限绑定 |
| 初代牧首 | **赫利奥（Helio）**，即城主本人（同一人，第 3 层身份） |
| 瓦解 | 修复停滞、人员凋零或转入休眠后，教团组织消散；塔下城市（渊光城）与塔体一并沉寂 |
| 遗存 | 暗渊口述中的「灯火」来自塔下残存供能与自动维护；少数后裔血脉/记录或已断绝（待定是否留个别 NPC） |

## 与骄阳会圣痕的关系

- 教团保存**圣痕**完整图谱，对接塔下 **佛勒贡（Phlegon）** 反应堆；色泽为 **蓝紫色**（见 [骄阳之心 · 反应堆色系与圣痕](../05-隐秘真相/骄阳之心.md#反应堆色系与圣痕)）。
- 覆灭时部分图谱经贸易与口述流入 [日生之地](../03-地点与场景/日生之地.md)，成为 [无敌骄阳会](./无敌骄阳会.md) 圣痕的失真来源之一——失真的不仅是纹样，还有色系（教团蓝紫 → 骄阳会 **埃同**橙黄金）。
- [无敌骄阳会](./无敌骄阳会.md) 史诗**自我闭环**，**不为渊光留位**；教团牧首传说与骄阳会赫利俄斯—普罗米修斯叙事在暗渊侧**同源异读**——同一历史人物被拆成两种互不收录的经卷。

## 玩法关联
