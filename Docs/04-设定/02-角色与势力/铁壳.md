# 铁壳（Qiào）

## 摘要

游荡于铁门关之外的机械体或装甲单位，击败后可掉落建材与燃料。与[无敌骄阳会](无敌骄阳会.md)敌对，但不会主动靠近铁门关。

## 玩家可见层

### 描述

- 游荡在铁门关之外的神秘机械体或装甲单位。
- 外形似是被遗弃的机械，外壳锈迹斑斑但仍有活动能力。
- **关键特性**：被太阳照射后会恢复活力/变得活跃，因此它们**不会主动靠近铁门关**（铁门关位于日生之地内，处于太阳照射范围）。
- 击败后可以回收建材和燃料，是荒地中重要的资源来源之一。
- 与[无敌骄阳会](无敌骄阳会.md)持续敌对，双方在荒地中时有冲突。
- 铁门关的军队必须时刻防范铁壳的袭击，无法抽调兵力执行其他任务。

### 与玩法的关系

- **第二章**穿越[荒地](../03-地点与场景/荒地.md)时的常见遭遇单位。
- 击败后掉落建材与燃料，补充长途跋涉的资源消耗。
- 与[无敌骄阳会](无敌骄阳会.md)敌对，玩家可利用这一关系（待确认具体玩法）。
- **不会主动靠近[铁门关](../03-地点与场景/铁门关.md)**：因为铁门关位于日生之地（太阳照射范围），被太阳照射会使铁壳恢复活力/变得危险。
- **第一章剧情关联**：铁门关军队因需防范铁壳，无法全力阻拦循烬城。
