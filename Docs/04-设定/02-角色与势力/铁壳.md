# 铁壳（Qiào）

## 摘要

游荡于 [铁门关](../03-地点与场景/铁门关.md) 之外的机械体势力。总部 [铁巢](../03-地点与场景/铁巢.md)，由远古智能 **巢主** 统领；巢主依赖**第二颗骄阳之心**存续。与 [无敌骄阳会](./无敌骄阳会.md) 永久敌对。

## 玩家可见

- 击败可获建材与燃料。
- 被太阳照射后会恢复活力/更危险。

## 玩法关联

- **第二章核心**：穿越 [铁巢废墟](../03-地点与场景/铁巢废墟.md) 遭遇巡逻 → 建立关系 → 抵达 [铁巢](../03-地点与场景/铁巢.md) 与巢主对峙。
- **第二颗骄阳之心**：若非敌对，面见巢主时直接赠与；否则需攻略铁巢抢夺（见 [铁巢](../03-地点与场景/铁巢.md)）。
