## 势力关系速查

| 势力A | 关系 | 势力B | 说明 |
|-------|------|-------|------|
| [城邦同盟](城邦同盟.md) | 相对对立 | [诸王国土](诸王国土.md) | 可交涉、贸易与暂时结盟；非永久敌对 |
| 无敌骄阳会 | 敌对 | 铁壳 | 铁壳游荡于荒地，与骄阳会武装冲突 |
| [猎壳人](猎壳人.md) | 狩猎 | 铁壳 | 关外以猎杀铁壳为业 |
| 玩家（循烬城） | 可选站队 | [猎壳人](猎壳人.md) / 铁壳 | 第二章荒地：协助猎壳人或与铁壳交好，影响双向关系（见 [章节划分与故事大纲](../05-隐秘真相/章节划分与故事大纲.md)） |
