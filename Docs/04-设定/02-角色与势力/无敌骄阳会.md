# 无敌骄阳会

## 摘要

[太阳城（骄阳之城）](../03-地点与场景/太阳城.md)的武装与统治组织，由 [骄阳之子](./骄阳之子.md)领导。与 [铁壳](./铁壳.md) 永久敌对。

## 玩家可见

- 太阳城周边巡逻的武装/宗教组织；对局外人傲慢但有秩序。
- 关外狩猎部队见 [猎壳人](./猎壳人.md)。
- 与铁壳在 [荒地](../03-地点与场景/荒地.md) 持续冲突；视与铁壳交好者为**亵渎**。
- 高阶成员经**受礼**仪式获得**圣痕**，用以承载、驱使 [骄阳之心](../05-隐秘真相/骄阳之心.md) 的力量——色泽对应该城反应堆专名色系（骄阳会本部以 **埃同 Aethon** 的橙黄金为主）；组织内以宗教仪式解释。
- 教义见下文 **§教义与史诗**；母本层与 [骄阳之心 · 四颗反应堆专名](../05-隐秘真相/骄阳之心.md#四颗反应堆专名已定) 对齐。

## 教义与史诗（玩家可见）

骄阳会正典是一部**自我闭环**的日神史诗：**天上太阳、日生之地、赠日之恩**构成完整世界，**不**包含 [渊光](../03-地点与场景/渊光.md) / 暗渊深处之塔或第四颗「分日」之外的隐秘。

### 赫利俄斯与普罗米修斯

| 位格 | 说明 |
|------|------|
| **赫利俄斯**（Helios） | **主崇**：一致高悬于空、庇护众生的太阳；礼拜与国策的核心 |
| **普罗米修斯**（Prometheus） | **先知者**：史诗中将太阳**分成两块**，把其中一块**交予** [日生之地](../03-地点与场景/日生之地.md) 地上之人者 |

**赠日叙事（信众口径）**：大崩溃之后，先知普罗米修斯分日予人，使人得以在光下生存；此举得赫利俄斯**应允**。地上的人感激普罗米修斯，**自发**为他建神殿，并以类似现实中歌颂基督的方式歌颂他——但教会礼仪与官方神学**更歌颂赫利俄斯**，因永悬之阳才是日常庇护；普罗米修斯是「恩赐予人者」，赫利俄斯是「恩所自出者」。

### 史诗与历史（母本）

| 信众说法 | 母本层对照 |
|----------|------------|
| 赫利俄斯永悬 | 人造太阳仍运转（急停后的「当前模式」） |
| 普罗米修斯分日 | 口述对「多颗骄阳之心 / 太阳被拆分」的宗教化；**不知**反应堆专名与 [指挥塔](../05-隐秘真相/指挥塔的真相.md) |
| **Helio / 赫利奥** | 紧急期当事人；专名与 **Helios** 音近，后世易被叠入日神叙事；信众**不知**即循烬城城主 |

城主 **赫利奥**知史诗为自身经历的误读与拼合，**拒绝**神格；史诗**刻意不留** [渊光城](../03-地点与场景/渊光城.md) 位置——信众若闻「暗渊深处的光」，多视为异端或野史。

### 史诗闭环与盲区（设定约束）

- **闭环范围**：赫利俄斯—普罗米修斯—日生之地—太阳城；「分出去的那一块太阳」足够解释骄阳之心与追日信仰。
- **刻意缺失**：[渊光](../03-地点与场景/渊光.md)、**佛勒贡（Phlegon）**、指挥塔、暗渊最深处之「第四块」——**不在**骄阳会经文中。
- **叙事用途**：第三章后玩家「向渊光」与骄阳会世界观**正面摩擦**；铁巢 **埃厄斯（Eos）** 亦在史诗外围（关外异端/铁壳叙事）。

## 圣痕（玩家可见）

- **外观**：线刻纹身；色泽随所对接的骄阳之心专名而定（见 [骄阳之心 · 反应堆色系](../05-隐秘真相/骄阳之心.md#反应堆色系与圣痕)）。**骄阳会本部**高阶成员以 **埃同（Aethon）** 的**橙黄金**为主，与主崇赫利俄斯一致。
- **位置**：高阶成员多可见于**颈侧与面部**（具体位置因阶位而异）；经受礼仪式授予。
- **功能（组织说法）**：承载骄阳之心的力量，并在受礼者体内「驱使」其威能；是骄阳会高阶身份与太阳亲和的标志。
- **与普罗米修斯**：圣痕被视为「恩行」在世间的印记；城主前臂+胸口的圣痕若暴露，可能被联想为古式或异色（**皮洛伊斯**铂白金系）。
- **母本层**：实为方舟人体—能源接口科技的一种后世沿袭；技术本质见 [骄阳之心 · 圣痕](../05-隐秘真相/骄阳之心.md#圣痕)。
