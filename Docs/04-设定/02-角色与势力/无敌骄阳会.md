# 无敌骄阳会

## 摘要

[太阳城（骄阳之城）](../03-地点与场景/太阳城.md)的武装与统治组织，由 [骄阳之子](城市领袖/骄阳之子.md)领导。与 [铁壳](./铁壳.md) 永久敌对。

## 玩家可见

- 太阳城周边巡逻的武装/宗教组织；对外人傲慢但不至于无礼。
- 关外狩猎部队见 [猎壳人](./猎壳人.md)。
- 与铁壳在 [荒地](../03-地点与场景/荒地.md) 持续冲突；视与铁壳交好者为**亵渎**。
- 正典为 **[《予日之诗》](../03-关卡与叙事/01-文献/予日之诗.md)**。
