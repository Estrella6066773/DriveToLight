# 无敌骄阳会

## 摘要

[太阳城（骄阳之城）](../03-地点与场景/太阳城.md)的武装与统治组织，由骄阳之子（教主+皇帝）领导。掌控**第三颗骄阳之心**（见 [骄阳之心](../05-隐秘真相/骄阳之心.md)）。与 [铁壳](./铁壳.md) 永久敌对。

## 玩家可见

- 太阳城周边巡逻的武装/宗教组织；对局外人傲慢但有秩序。
- **猎壳人**：骄阳会派往 [荒地](../03-地点与场景/荒地.md) 的狩猎队伍，以猎杀 [铁壳](./铁壳.md) 为业；在关外设前出营地（多刷新于 [村镇](../03-地点与场景/荒野地块/村镇.md) 类格子）。
- 与铁壳在荒地持续冲突，原因不明；视与铁壳交好者为**亵渎**。

## 玩法关联

- **第一章**：封锁循烬城欲夺第一颗骄阳之心 → 拦不住 → 点明循烬城有骄阳之心后放行。
- **第二章**：
  - 关外以猎壳人队伍与前出营地形态出现；协助猎杀铁壳可获大量建材、燃料，营地可对玩家开放。
  - **站队**：玩家若重点与铁壳交好，骄阳会视为亵渎，关系恶化（详见 [章节划分与故事大纲 · 猎壳人与铁壳](../05-隐秘真相/章节划分与故事大纲.md#猎壳人与铁壳站队分支)）。
- **第四章**：[铁门关](../03-地点与场景/铁门关.md) 冲突后赢得尊重与放行。
- 城市详情见 [太阳城](../03-地点与场景/太阳城.md)；章节见 [章节划分与故事大纲](../05-隐秘真相/章节划分与故事大纲.md)。
