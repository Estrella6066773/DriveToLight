| **何时新增** | 系统设计文档 `校验状态` 为 **待对齐** / **部分符合**，且问题属规则或流程而非程序实现或设定母本时 |

| **当前开放项** | **只写未解决**；不写「已定」复述。规则正文见 [02-系统设计](../02-系统设计/) 对应文档，本表只链过去 |
| **闭环** | 问题全部解决 → 移入「已解决记录」→ 从开放项删除该行 → 修订记录留痕 |
| **交叉项** | 主轨写全文；另一轨在「交叉引用」节链回，**不重复**整张表行 |
| **讨论稿** | 过程放在 [01-草稿/](../01-草稿/)，本文件只记缺口 |
