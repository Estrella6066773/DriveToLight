# 交互链：循光之城（第三、四章 · 指定目标：渊光）

## 在本作抽象中的位置

全篇交互链为 **向指定目标前进 + 六块**（见 [核心抽象 · 六块](./交互链-循光之城.md#六块当前位置经营)）。本篇在①块展开 **指定目标为渊光**；②～⑥见本章详图与全篇同构。

| 包含 | 不包含 |
|------|--------|
| 指定目标为渊光时的①块展开 | 指定目标为太阳（一二章） |
| 第三章入暗渊转向、第四章返程救援 | 第五章指挥塔终局供能细节 |
| 全局暗渊带、太阳移动停用口径 | 日照带、速度差追日 |

## 图例（与全篇版一致）

| 类型 | 含义 |
|------|------|
| **目标** | 玩家要达成什么 |
| **行为** | 玩家主动做什么 |
| **障碍** | 卡住、失败或需克服的状态 |
| **奖励** | 资源、情绪收益等较持久的正向结果 |
| **反馈** | 行为后的即时正向结果 |
| **决策信息** | 支撑判断的信息、分支与心算维度 |

> **拓扑**：**只向下分散、不向下合并**。
