## 3. 流程图

### 3.1 周总结总流程

```mermaid
flowchart TD
    EnvDone[环境行动结束 turn%7==0] --> RunSW[WeeklySummaryOrchestrator.Run]
    RunSW --> Steps[按 Module 顺序执行 Steps]
    Steps --> Split[Resource: 逐 mobile_city_id]
    Split --> Draw[FoodWarehouseDrawdown 仓库扣减]
    Draw --> Override[RunFoodOverrides]
    Override --> Alloc[L1城区占比 L2归属占比 L3同比例]
    Alloc --> Fed[分到者存活]
    Alloc --> Unfed[未分到者]
    Unfed --> Death["死 floor(n x 50%)"]
  Death --> Trim[TrimResidentsAndTeamsAfterLoss]
    Fed --> Delta[ApplyWorkHeadcountChange]
    Death --> Delta
    Trim --> Delta
    Delta --> Next[下一回合玩家指挥]
    Ruins[废墟] --> Alloc
    Ruins --> Block[永不可工作]
```
