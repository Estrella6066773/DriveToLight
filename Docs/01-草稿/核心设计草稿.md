# 核心设计草稿：延续

## 一句话点子

驾驶一座可拆解重组的模块化移动城市，在末日废土上追逐移动的人造太阳，在资源、速度与牺牲之间做出艰难抉择，以求文明延续。

## 为什么吸引我

- **情绪张力**：追逐太阳本身就带有强烈的方向感与宿命感，结合后启示录的荒凉，形成一种“明知艰难却必须前行”的悲壮。
- **策略深度**：城市可以分离、抛弃、重建，让“取舍”不只是数字，而是真实切割自己领土的痛楚。
- **视觉潜力**：末日废墟与太阳朋克科技并置，绝望中透出生机，画面辨识度高。
- **悬疑钩子**：表层是追日之旅，底层是“培养皿/飞船”的真相，给长线叙事留下反转空间。

## 世界观与故事背景

### 核心设定

游戏发生在文明覆灭又重生的后启示录世界。天空中的“人造太阳”原本固定不动，现在开始移动。玩家扮演一座移动城市的领袖，带领幸存者追逐太阳，抵达它所在之处，以延续文明。

### 叙事主题

- 生存不是终点，抵达太阳才是目的。
- 前进本身即是意义；停滞意味着被黑暗与严寒吞没。
- 领袖的抉择塑造城市的历史，也会铭刻在牺牲者的记忆里。

### 秘密背景（后期悬念）

整个世界的真相可以隐喻为“一个巨大飞船或培养皿内部”。文明如同意外被放在酒精灯上烤了一晚、却仍存活的菌群。这层真相不会在游戏前期揭示，只作为远景悬疑存在，服务于终局反转。

## 核心玩法与体验

### 游戏类型

侧重于资源管理与策略规划的模拟经营游戏。

### 核心体验

- **规划与取舍**：每一次扩建、分离、加速，都需要权衡当前需求与未来风险。
- **动态压迫**：离太阳越远，环境越严酷，迫使玩家不断向太阳靠近。
- **模块割舍**：城市不是单一实体，可以被切割、牺牲、重组，让战略决策带有情感重量。

### 胜利条件

最终目标是驾驶城市抵达人造太阳的所在地。游戏鼓励甚至迫使玩家持续向太阳移动。

### 动态难度

- 停滞不前或离太阳过远时，生存难度会不断加大。
- 具体表现可包括：资源产出下降、灾害频率提升、人口消耗增加、士气/希望值下降等。
- 越靠近太阳，环境越温和，但竞争/稀缺性也可能随之变化。

## 关键机制与设计

### 地图与移动

- 世界地图由正六边形格子组成。
- 玩家的主城占据多个格子，每个格子视为一个“城区”。
- 城市可以整体移动，但主城与外界的交互通常需要停下来才能进行。
- 特殊地形（如裂谷）可能需要城市拆分模块、分批通过后重组，这既是策略挑战，也是一种资源/速度的牺牲。

### 三种核心资源

> 注：文档前半部分提到三种资源，后续补充为四种。当前以四种为准，保留三种作为早期简化方向。

基础资源包括：

| 资源 | 主要用途 | 典型来源 |
|------|----------|----------|
| 建材 | 建造/修复城区、采掘站、驿站等 | 矿区、废墟、城市 |
| 食物 | 维持人口消耗、远征队补给 | 果地、废墟、村镇、城市 |
| 燃料 | 驱动城市移动、维持能源需求 | 矿区、废墟、城市 |
| 人口 | 分配工作、组建队伍、城市运转 | 村镇、城市 |

### 探索与扩张

- 移动到新地区后，可组织远征队进行勘探。
- 可建立采集站、驿站等设施。
- 可前往固定据点补充人口与资源。
- 默认情况下，村镇和城市不能被工程队开采。

### 城市模块化

- 主城中心为“核心区”。
- 与核心区及已连接区域相连的格子才算作城市的有效城区。
- 城区可以从城市中分离，被抛弃或拆解回收资源。
- 城区可以在空地新建，再连接回城市。
- 拆除后的区域可以像资源点一样进行能量回收。

### 城区类型（当前规划）

| 类型 | 定位 |
|------|------|
| 核心区 | 主城中心，城市存在的根基 |
| 居民区 | 提供人口承载与基础生活支持 |
| 工业区 | 提升建材/燃料等生产与加工效率 |
| 接驳区 | 支持模块分离、重组与对外连接 |

后续系统待完善。

## 队伍系统

队伍分为以下类型，后续可能扩展战斗相关队伍：

| 队伍类型 | 职责 |
|----------|------|
| 勘探队 | 外出探索并点亮视野，发现资源点与地形信息 |
| 运输队 | 在各个格子之间运输物资 |
| 工程队 | 开发和运营资源地块；默认不能开采村镇和城市 |

## 荒野地点

城市之外的地区视为荒野，荒野格子上会刷新各类地点：

| 地点 | 产出 |
|------|------|
| 矿区 | 建材 / 燃料 |
| 果地 | 食物 |
| 废墟 | 建材 / 燃料 / 食物 |
| 村镇 | 食物 / 人口 |
| 城市 | 建材 / 燃料 / 食物 / 人口 |

## 可能的延伸

- 人造太阳移动规律：固定周期移动？随机事件驱动？受玩家行为影响？
- 多城市/势力：废土上是否还有其他移动城市？可交易、竞争或吞并？
- 事件与叙事卡：在移动过程中触发随机事件，深化“绝望中求存”的氛围。
- 希望/士气系统：作为人口稳定性与生产效率的隐藏变量。
- 终局真相：抵达太阳后揭示“培养皿”隐喻，提供多重结局。
