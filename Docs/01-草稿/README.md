# 草稿

本目录用于存放《循光之城》的灵感、脑暴、情绪板、参考图、快速涂鸦。

## 使用原则

- **不追求完美**：这里的文档允许凌乱、跳跃、中英文混杂。
- **快速记录**：想到就写，不必等结构完整。
- **定期收敛**：每周或每个里程碑，挑选有价值的内容整理进 `02-系统设计/` 或 `04-设定/`。

## 子目录建议

| 子目录 | 用途 |
|--------|------|
| `情绪板/` | 参考图、配色、氛围截图 |
| `灵感碎片/` | 一句话点子、机制雏形 |
| `参考游戏/` | 竞品分析、可借鉴的设计 |
| `会议记录/` | 语音/文字讨论纪要 |
