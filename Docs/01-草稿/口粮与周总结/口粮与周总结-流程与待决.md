## 3. 流程图

### 3.1 周总结总流程

```mermaid
flowchart TD
    EnvDone[环境行动结束 turn%7==0] --> RunSW[WeeklySummaryOrchestrator.Run]
    RunSW --> Steps[按 Module 顺序执行 Steps]
    Steps --> Split[Resource: 逐 mobile_city_id 口粮]
    Split --> Pool[单元内: 城市池 + 载荷口粮]
    Pool --> Demand[需求 = 活跃人口 x 1]
    Demand --> Alloc[FoodAllocator 在岗优先]
    Alloc --> Fed[分到: 解除饥饿]
    Alloc --> Unfed[未分到]
    Unfed --> WH{上周期已饥饿?}
    WH -->|是| Death["死 floor(n x 50%)"]
    WH -->|否| Hunger[标记饥饿]
    Hunger --> Delta[ApplyWorkHeadcountChange]
    Death --> Delta
    Fed --> Delta
    Delta --> Next[下一回合玩家指挥]
    Ruins[废墟] --> Demand
    Ruins --> Block[永不可工作]
```
