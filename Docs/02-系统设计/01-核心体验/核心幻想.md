## 一句话描述

驾驶一座可拼接、可拆解的模块化移动城市，在太阳逐渐的永昼世界里追逐太阳。体验亲手建造设施，又被迫将其放弃的求生之旅。

## 关键词

- 属于太阳朋克的后启示录
- 移动城市 / 模块化 / 取舍
- 追逐太阳 / 前进 / 延续
- 绝望 / 希望 / 牺牲

## 玩家目标

- **短期目标**：维持[四种核心资源](../04-资源与人口/四种核心资源.md)平衡，让城市在当前区域生存并补给。
- **中期目标**：通过[探索与扩张](../07-玩法循环/探索与扩张.md)点亮荒野、建立采集设施与驿站，扩张或重组城区以应对前方地形。
- **长期目标**：驾驶城市抵达太阳所在地，延续文明；揭示世界隐藏设定（长期悬念）。

## 情绪曲线

玩家在持续前进中经历「资源紧缺的压迫 → 探索发现的短暂希望 → 地形或取舍带来的悲壮 → 靠近太阳时的缓和与新的稀缺」循环。类型与视觉、情感基调见 [核心世界观 · 世界风格与基调](../../04-设定/01-世界观/核心世界观.md#世界风格与基调)。
