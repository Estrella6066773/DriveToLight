# 系统设计

> 对外读者请先从 [游戏介绍](../游戏介绍.md) 入手；本目录各文档为机制专题详解。

## 设计文档模板

新建设计文档请使用 [系统设计文档模板](../_templates/系统设计文档模板.md)。

正文首次提及其它系统设计文档或已在库中单独立篇的专有名词时，须链到对应文件（同一文档内同一目标仅链接一次）。详见 [文档库使用说明 §6](../00-规范/文档库使用说明.md#6-正文交叉链接对外展示文档)。

涉及可执行能力、数值参数、队伍模板的设计条目，统一遵循 [SO 配置与能力通道规范](../00-规范/SO配置与能力通道规范.md)。
