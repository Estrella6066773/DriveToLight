# 系统设计

本目录存放《循光之城》的**玩家体验、玩法规则、流程、数值设计意图**。

> 对外读者请先从 [游戏介绍](../游戏介绍.md) 入手；本目录各文档为机制专题详解。

| [01-核心系统/](01-核心系统/) | 核心玩法、地图移动、队伍、探索扩张 |
| [02-玩法循环/](02-玩法循环/) | 分钟级/小时级/长期循环 |
| [02-资源循环/](02-资源循环/) | 资源类型、荒野地点、产出与消耗 |
| [03-模块与城市/](03-模块与城市/) | 城市模块化、城区类型、连接规则 |
| [03-关卡与叙事/](03-关卡与叙事/) | 关卡结构、任务流程、叙事节奏 |
| [04-数值框架/](04-数值框架/) | 数值目标、成长曲线、经济模型 |
