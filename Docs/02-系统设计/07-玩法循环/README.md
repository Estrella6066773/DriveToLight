# 玩法循环

本目录定义《循光之城》的**时间尺度与回合推进**：玩家在不同粒度下做什么、每回合如何结算。

```mermaid
flowchart LR
    A[核心循环] --> B[回合与行动表]
    B --> C[工作]
    C --> D[探索与扩张]
```

| 顺序 | 文档 | 回答的问题 |
|------|------|------------|
| 1 | [核心循环](核心循环.md) | 玩家在短期/中期/长期三级尺度上做什么 |
| 2 | [回合与行动表](回合与行动表.md) | 回合如何推进、指挥与行动阶段 |
| 3 | [工作](工作.md) | 跨回合工作的进度与成果 |
| 4 | [探索与扩张](探索与扩张.md) | **停泊后**的勘探、建设、运输、据点交互（子循环） |

## 时间尺度与回合的对应

| 时间尺度 | 典型跨度 | 玩家在做什么 | 回合机制 |
|----------|----------|--------------|----------|
| **短期** | 数回合内 | 观察资源、调整编制、应对即时事件 | [玩家指挥](回合与行动表.md#玩家指挥阶段的操作) |
| **中期** | 十数回合 | 选方向、组织勘探、开发资源、补给 | [指令表与自主执行](回合与行动表.md#指令表与自主执行) |
| **长期** | 整局 | 追逐太阳、扩张能力、推进叙事 | [环境行动](回合与行动表.md#环境结算顺序) |

**探索与扩张**是「中期循环」的**子循环**：城市停泊后，玩家派出队伍进行勘探、建设、运输等操作，完成后再次移动。
