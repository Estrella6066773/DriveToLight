## 目标

定义城市由多个可拆分、重组、牺牲的城区组成的核心规则，使“取舍”成为玩家能清楚感受到的策略选择。

## 范围

- **包含**：城区定义、连接规则、分离/拆解/新建规则、城区类型。
- **不包含**：建造消耗、模块数值、UI 交互流程。

## 详细说明

### 术语

- **[移动城市](../01-核心系统/地图与移动.md)**即**玩家主城**；地图上仅此一座可整体移动。
- 荒野上的城市、村镇等为**[外部据点](../02-资源循环/荒野地点.md)**，不属于移动城市。

### 核心区

- 核心区是移动城市的核心，城市依赖它存在与运转。
- 与核心区相连、且经连续连接可达的格子，才算作城市的有效城区。

### 连接与分离

- 城区可以从城市中分离，被抛弃或拆解回收资源。
- 城区可以在空地新建，再连接回城市。
- 拆除后的区域可以像资源点一样进行资源回收。
- **分离规则**：若城市的城区被拆解成两部分，凡不能连接到核心区的城区，一律从城市中分离。

### 城区类型（当前规划）

| 类型 | 定位 |
|------|------|
| 核心区 | 移动城市的核心；失去连接的有效城区将被分离 |
| 居民区 | 提供人口承载与基础生活支持 |
| 工业区 | 提升建材/燃料等生产与加工效率 |
| 接驳区 | 支持模块分离、重组与对外连接 |

后续系统待完善。
