## 目标

定义城市由多个可拆分、重组、牺牲的城区组成的核心规则，使“取舍”成为可感知的策略行为。

## 范围

- **包含**：城区定义、连接规则、分离/拆解/新建规则、城区类型。
- **不包含**：建造消耗、模块数值、UI 交互流程。

## 详细说明

- 主城中心为“核心区”。
- 与核心区及已连接区域相连的格子才算作城市的有效城区。
- 城区可以从城市中分离，被抛弃或拆解回收资源。
- 城区可以在空地新建，再连接回城市。
- 拆除后的区域可以像资源点一样进行能量回收。

### 城区类型（当前规划）

| 类型 | 定位 |
|------|------|
| 核心区 | 主城中心，城市存在的根基 |
| 居民区 | 提供人口承载与基础生活支持 |
| 工业区 | 提升建材/燃料等生产与加工效率 |
| 接驳区 | 支持模块分离、重组与对外连接 |

后续系统待完善。
