## 目标

定义荒野格子上**资源层**资源点的类型、自然形态、储量产出，以及可在其上建立的**采集设施**；与**建筑层**上的**城区**区分。

## 范围

- **包含**：四类资源点、自然形态 → 采集设施对应、储量与工作转化、开采限制；建筑层**城区**（含废墟状态）的交叉引用。
- **不包含**：刷新频率、具体产出数值、采集设施维护公式。

## 详细说明

城市之外的 [荒地](../../04-设定/03-地点与场景/荒地.md) 格子上可刷新**资源层**内容。每格可有**储量**；工程队可在合法资源点上建立对应**采集设施**，通过 [工作](../07-玩法循环/工作.md) 将储量转化为 [物品层](../03-图层与地点/地图图层.md) 中的金属、食物、能源或人口相关资产。

**采集设施与储量参数**：[L3_resource_defs.csv](../../03-程序设计/03-数据字典/tables/L3_resource_defs.csv)（一行一类；白名单列覆盖地形档位）。

### 资源点与采集设施

| 资源点 | 自然形态（玩家可见） | 储量产出 | 采集设施（设施层，建于该资源点） | 设定词条 |
|--------|----------------------|----------|----------------------------------|----------|
| **村镇** | 小型聚落外观；**机制上就是资源点** | **无归属人口**（储量形态 **待定**） | **征兵办** | [村镇](./荒野地点/村镇.md)、[征兵办](./荒野地点/征兵办.md) |
| **矿藏** | **裸露金属结构** | **金属** | **矿区** | [矿藏](./荒野地点/矿藏.md)、[矿区](./荒野地点/矿区.md) |
| **果地** | **野生果林** | **食物** | **果园** | [果地](./荒野地点/果地.md)、[果园](./荒野地点/果园.md) |
| **遗迹** | **前文明遗迹**：颇具辨识度的远古结构 | **能源** | **能源站** | [遗迹](./荒野地点/遗迹.md)、[能源站](./荒野地点/能源站.md) |

- **遗迹**为**不可重建**的资源点：本体不能修复为城区或建筑，只能在合法条件下建立 **能源站** 开采储量。

- **村镇（已定）**：**就是**资源层资源点，与矿藏 / 果地 / 遗迹**同一套**交互——揭示 → 勘探 → 工程队建 **[征兵办](./荒野地点/征兵办.md)** → [工作](../07-玩法循环/工作.md) 转化储量；**无**贸易、领袖页、商队或组织传导等**独立据点**玩法。储量 **无归属**人口；提取前**不计**势力粮食需求；提取后归属见 [领袖与势力](../05-城市与领袖/领袖与势力.md)、[人口与迁移](./人口与迁移.md)。详见 [村镇 · 资源点交互](./荒野地点/村镇.md#资源点交互已定)。
- **征兵办**、**矿区**、**果园**、**能源站**均属 [设施层 · 占格类 · 生产类（储量转化）](../03-图层与地点/设施层.md#占格类--三类细分)：**建立于资源点**，有效时提供工作与产出；建造与修复消耗 **金属**，部分持续运行消耗 **能源**（见 [四种核心资源](./四种核心资源.md)）。

### 建筑层城区

[建筑层](../03-图层与地点/地图图层.md#建筑层与城区) 存储的就是**城区**（与城区划等号），不是与城区并列的另一套类型：

| 内容 | 说明 | 文档 |
|------|------|------|
| **城区**（状态：**正常**） | 特殊 / 一般类别；可承载人口（若允许）；可接入移动城市 | [城市模块化](../03-图层与地点/建筑层/README.md) |
| **城区**（状态：**废墟**） | 被击毁或**默认刷新**的损毁城区；玩家须能**认出**；**结构完整度 低于 20%** 强制转废墟；**无法工作**；**不可迁入**、仅可**迁出**居民 | [城市模块化 · 废墟](../03-图层与地点/建筑层/城区总览.md#废墟) · [废墟](./荒野地点/废墟.md) |
| **外部城市** | 固定多格据点；**城市领袖**为领袖关系主体；由若干城区占格组成；不可整体移动 | [城市-外部](./荒野地点/城市-外部.md)、[势力系统](../05-城市与领袖/势力系统.md) |

### 遗迹与废墟（术语强制区分）

| | **遗迹** | **废墟**（城区状态） |
|---|----------|----------|
| **图层** | [资源层](../03-图层与地点/地图图层.md) | [建筑层](../03-图层与地点/地图图层.md)（**城区**实例） |
| **本质** | **前文明遗迹** | **状态为废墟的城区**（被击毁或默认刷新） |
| **可否重建** | **不可重建**（仅能建采集设施开发储量） | **须修复**为状态**正常**的城区后，机制上才按正常城区处理（消耗资源 + 工作 **待定**） |
| **主要交互** | 建立 **能源站**，工作转化能源储量 | 修复、拆解、回收；修复完成前**不**按正常城区结算 |
| **与人口** | 不承载人口 | **不可迁入**；已有居民**仅可迁出**；**无法工作**；**计**粮食需求（修复为正常前） |
| **玩家辨识** | 与前文明遗迹外观区分（**待定**细节） | 须能**认出**废墟，并与正常城区、遗迹区分 |

> 命名地区 [铁巢废墟](../../04-设定/03-地点与场景/铁巢废墟.md) 为剧情地区专名，**不是**网格上的城区状态「废墟」，也不是资源层「遗迹」。

### 资源点揭示机制

权威定义见 [单位类型与视野 · 资源点揭示等级](../06-单位与交战/单位类型与视野.md#资源点揭示等级)（**隐藏 / 揭示种类 / 揭示储量**三级，程序值 `hidden` / `kind` / `amount`）。

**探索流程**：

1. 侦察队将资源点提升至 **揭示种类**（`kind`）。
2. 勘探队经 **50% / 100%** 勘探工作提升至 **揭示种类** / **揭示储量**（`amount`）。
3. 据储量与距离，派出 [运输队](../06-单位与交战/单位类型与视野.md#3-运输队模板) 与 [工程队](../06-单位与交战/单位类型与视野.md#4-工程队模板) 建立采集设施并运转。

### 开采与交互限制

- 上表四类资源点（**含村镇**）：工程队可在满足 [地图图层](../03-图层与地点/地图图层.md) 规则处建立对应**采集设施**；交互流程见 [资源点揭示机制](#资源点揭示机制) 与 [村镇 · 资源点交互](./荒野地点/村镇.md#资源点交互已定)。
- **外部城市**格：不可当作资源点开采；交互见 [探索与扩张 · 外部城市](../07-玩法循环/探索与扩张.md#外部城市交互)。
- **废墟**格（**状态为废墟的城区**）：**不**当作资源点，**不**建立矿区 / 果园 / 能源站；走城区修复、拆解规则。
- **遗迹**格：**不可重建**本体；仅可建 **能源站** 开发能源储量。
