# 温室

## 摘要

建于 **一般城区设施建造位** 的 **占格类 · 生产类**设施：消耗 **能源**，产出 **食物**。用于后期 **暗渊** 等无果地 / 自然资源稀缺区域，以 **1～2 座** 维持全城粮食。

## 玩家可见

封闭培植舱、补光与温控装置；须工作区开启且运作人口上岗（与一般城区设施同逻辑，见 [运作与居民](../03-图层与地点/建筑层/运作与居民.md)）。

## 转化产量（已定）

对照锚：

| 锚 | 数值 |
|----|------|
| 后期城规模（例） | **7** 城区 → 满编周需约 **350** |
| 单次产出周期 | **3 回合**（与果园同） |
| 暗渊配置 | **1～2** 座温室 |

| 参数 | 首版默认 | 说明 |
|------|----------|------|
| 工作类型 | `convert_energy_to_food` | 对设施实例下达生产工作，首次下达后每周期完成自动延续循环至玩家主动中断；食物类——**中断即从头开始**（不可从暂停进度继续） |
| `required_work_amount` | **30** | 对照：满编工程队或等价运作人力专精 **3 回合** |
| `output_per_work_completion` | **150** 食物 | 强力后期设施 |
| `input_cost_json` | `{"energy":50}` | 每次完成时扣 **50** 能源；不足则**拒绝完成** / 不结算产出 |
| 折合周产（专精 1 座） | 约 **350**/周（7÷3 × 150） | 约等于 **7** 满编区周需 |

**后期平衡（7 城区满编 ≈ 350/周）**

| 温室数 | 专精周产约 | 相对周需 350 |
|--------|------------|--------------|
| **1** | ~350 | 满编紧平衡（须专精运转 + 稳定能源） |
| **2** | ~700 | 宽裕；可囤粮、扩城或承受停机 |

**农业词条**：归属城带 **农业** 时，食物产出 **×1.3**（150 → **195**）；能源消耗**不**减免（除非另有领袖 / 能力，**待定**）。

建造对照时长 **3** 回合；建造金属 **80**（见 [产出基准](./四种核心资源.md#四类资源产出基准已定)）；须 **一般城区** 设施位。
