# 能源站

## 摘要

建于 **[遗迹](./遗迹.md)**（**前文明遗迹**资源点）上的 **采集设施**；将能源储量转化为可用 **能源**。**遗迹本体不可重建**，仅维护/扩建采集设施。

## 玩家可见

接入远古遗迹结构的供能设施与维护通道；由 [工程队](../../06-单位与交战/队伍系统.md) 建造与运维。

## 采集产量（已定 · 对齐运作与温室）

| 参数 | 首版默认 | 说明 |
|------|----------|------|
| 工作类型 | `extract_reserve` | 玩家首次下达采集，此后每周期完成自动延续循环至玩家主动中断；非食物类——**可暂停不重置**（进度保留，恢复后从暂停处继续） |
| `required_work_amount` | **30** | 满编工程队专精 **3 回合** |
| `output_per_work_completion` | **40** 能源 | 写入城市能源池 |
| 折合周产（专精） | 约 **93**/站/周 | 7÷3 × 40 |
| 建造 | **30** 金属；对照 **3** 回合 | 见 [产出基准](../四种核心资源.md#四类资源产出基准已定) |

**供需对照（专精）**

| 用途 | 约周耗 | 约需能源站 |
|------|--------|------------|
| 核心区日常（不可关） | ~35 | — |
| 核心 + **1** 温室 | ~152 | **2** 座紧平衡 |
| 上项 + **2** 温室 | ~269 | **3** 座 |

遗迹储量：`reserve_min=500`，`reserve_max=900`（约 12～22 次满额采集）。
