# 矿区

## 摘要

建于 **[矿藏](./矿藏.md)** 资源点上的 **采集设施**，将储量转化为 **金属**。

## 玩家可见

矿坑、选矿台与运输栈道等；由 [工程队](../../06-单位与交战/队伍系统.md) 建造与运维。

## 采集产量（已定 · 对齐修复）

| 参数 | 首版默认 | 说明 |
|------|----------|------|
| 工作类型 | `extract_reserve` | 玩家下达采集 |
| `required_work_amount` | **30** | 满编工程队专精 **3 回合** |
| `output_per_work_completion` | **25** 金属 | 与果园食物单次同量级 |
| 折合周产（专精） | 约 **58**/矿/周 | |
| 建造 | **30** 金属；对照 **3** 回合 | 见 [产出基准](../四种核心资源.md#四类资源产出基准已定) |

**供需对照**：航行负载每 **3** 格、每城区累积 **4** 金属当量损伤；**4** 城区航行 **12** 格 ≈ **64** 金属待修 → 约 **2** 座矿区专精一周可覆盖一趟中距航行修复（见 [金属与能源消耗基准](../四种核心资源.md#金属与能源消耗基准已定)）。

矿藏储量：`reserve_min=400`，`reserve_max=700`（与果地同档）。

## 玩法关联

- 负载成本 / 修复单价见 [城区总览 · 负载成本](../../03-图层与地点/建筑层/城区总览.md#负载成本)、[分离与拆解 · 修复](../../03-图层与地点/建筑层/分离与拆解.md#修复城区)。
