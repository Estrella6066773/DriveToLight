# 果园

## 摘要

建于 **[果地](./果地.md)** 资源点上的 **采集设施**，将储量转化为 **食物**。前期荒野补给主路径。

## 玩家可见

围护果林、晾晒场与简易仓房；由 [工程队](../../06-单位与交战/队伍系统.md) 建造与运维。

## 采集产量（已定）

对照锚：

| 锚 | 数值 |
|----|------|
| 城区基础住宅 | **50** |
| 周粮 | **1**/人 → 满编一区 **50**/周 |
| 前期城规模（例） | **4** 城区 → 周需约 **200** |
| 单次产出周期 | **3 回合** |
| 前期资源点数量 | **3～5** 座果园维持补给 |

| 参数 | 首版默认 | 说明 |
|------|----------|------|
| 工作类型 | `extract_reserve` | 玩家下达采集；**不**在建造完成时自动产出 |
| `required_work_amount` | **30** | 对照：工程队满编（10 × 效率 1.0）专精 **3 回合** |
| `output_per_work_completion` | **25** 食物 | 写入城市食物池 / 指定仓库 |
| 折合周产（专精） | 约 **58**/园/周（7÷3 × 25） | 与周总结扣粮分轨计时，按多周期均摊 |

**前期平衡（4 城区满编 ≈ 200/周）**

| 果园数 | 专精周产约 | 相对周需 200 |
|--------|------------|--------------|
| **3** | ~175 | 偏紧（须人口未满编、农业词条或第 4 点） |
| **4** | ~233 | 可维持并略有囤粮 |
| **5** | ~292 | 较宽裕，可支撑扩城过渡 |

**农业词条**：归属城带 **农业** 时，本次产出 **×1.3**（25 → **32.5**）。

建造对照时长 **3 回合**；建造金属 **30**（见 [产出锚](../四种核心资源.md#四类资源产出锚已定)）。

后期进入无果地的 **暗渊** 后，改以城内 [温室](../温室.md) 为主（见该文）；果园仍可用于有果地的区域。
