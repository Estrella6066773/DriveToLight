# 村镇

## 摘要

荒野 **资源层** **人口类资源点**：外观为小型聚落，**机制上就是资源点**——与矿藏、果地、遗迹**同一套**交互与开采流程；储量产出 **无归属**人口；采集设施为 **[征兵办](./征兵办.md)**。

## 玩家可见

规模小于 [城市-外部](./城市-外部.md) 的聚落外观。**不能**像外部城市一样贸易、接委托或打开领袖页；交互方式与矿藏 / 果地相同：侦察揭示 → 工程队建采集设施 → 队伍工作转化储量。

## 资源点交互（已定）

村镇**完全是** [资源点](../荒野地点.md#资源点与采集设施)，**不**存在单独的「据点交互」层：

| 步骤 | 与其它资源点 | 村镇特例 |
|------|--------------|----------|
| **揭示** | 侦察队进入视野后即时揭示全部信息 | 同左；见 [资源点揭示](../../06-单位与交战/单位类型与视野.md#资源点揭示) |
| **建采集设施** | 工程队在资源点格建立对应设施 | **征兵办**（占格类 · 生产类 · 储量转化） |
| **开采** | [工作](../../07-玩法循环/工作.md) 将储量转化为物品 / 人口 | 储量 **无归属**人口 → 经征兵办单次提取 **15** 人（见 [征兵办](./征兵办.md)） |
| **维护** | 建造消耗金属（征兵办 **30**）；首版无持续耗能 | 同左 |

## 储量（已定）

| 参数 | 首版默认 |
|------|----------|
| `reserve_min` | **60** 人 |
| `reserve_max` | **120** 人 |

约 **4～8** 次满额提取（15 人/次）。

- 提取前村镇储量**不计**势力粮食需求；提取后进入城区驻留或队伍 **随行人员** 时，由**接收城市的城市领袖**纳入管辖（见 [领袖与势力](../../05-城市与领袖/领袖与势力.md)、[人口与迁移](../人口与迁移.md)）。

## 玩法边界（已定）

- **非** [城市-外部](./城市-外部.md)；**非**领袖关系主体。
- **不**参与 [势力系统](../../05-城市与领袖/势力系统.md) 贸易、委托、组织间接传导或商队目标。
- 关系事件 **`home_city_ref`** **不**指向村镇格。
