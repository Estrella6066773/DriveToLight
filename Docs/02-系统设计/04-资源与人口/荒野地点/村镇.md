# 村镇

## 摘要

荒野 **资源层** **单纯**人口资源点：小型聚落；储量产出 **无归属**人口；**无**贸易、领袖页交互或组织传导等其它功能。

## 玩家可见

规模小于 [城市-外部](./城市-外部.md)；流浪者或小聚落外观。玩家在本地建立 [征兵办](./征兵办.md) 后，可从此资源点提取 **无归属**人口。

## 玩法关联

- 规则见 [荒野地点 · 村镇](../荒野地点.md#资源点与采集设施)。
- 采集设施：[征兵办](./征兵办.md)（**唯一**玩法入口）。
- **已定**：**非** [城市-外部](./城市-外部.md)、**非**领袖关系主体；**不**参与 [势力系统](../../05-城市与领袖/势力系统.md) 贸易、委托或组织间接传导；关系事件 **`home_city_ref`** **不**指向村镇格。提取后人口归属见 [领袖与势力](../../05-城市与领袖/领袖与势力.md)、[人口与迁移](../人口与迁移.md)。
