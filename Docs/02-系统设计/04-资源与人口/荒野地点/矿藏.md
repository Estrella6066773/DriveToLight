# 矿藏

## 摘要

荒野 **资源层** 矿物点：自然形态为 **裸露金属结构**，储量产出 **金属**。

## 玩家可见

地表或浅层的金属矿脉、残骸与矿化结构；需 [工程队](../../06-单位与交战/队伍系统.md) 建立 [矿区](./矿区.md) 后持续开采。
