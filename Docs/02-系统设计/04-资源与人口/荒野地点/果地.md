# 果地

## 摘要

荒野 **资源层** 食物点：自然形态为 **野生果林**，储量产出 **食物**。

## 玩家可见

野果林、菌菇地等；需建立 [果园](./果园.md) 后持续采集。

## 储量（已定 · 对齐果园 3 回合周期）

| 参数 | 首版默认 | 说明 |
|------|----------|------|
| `reserve_min` | **400** | 约 **16** 次满额采集（25/次）≈ 专精约 **7 周** |
| `reserve_max` | **700** | 约 **28** 次 ≈ 专精约 **12 周** |
| 设计意图 | 单点支撑前期一段航程后再须换点 | 与 [果园](./果园.md) `output=25` / 周期 3 回合对齐 |

刷新时在 `[reserve_min, reserve_max]` 内取值（首版均匀即可）。
