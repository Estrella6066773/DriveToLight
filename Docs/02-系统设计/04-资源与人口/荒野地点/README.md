# 荒野地点类型

← [荒野地点](../荒野地点.md) · [资源循环](../README.md)

荒野网格上的地点类型。玩法规则见 [荒野地点](../荒野地点.md)。

## 资源层（四类，可建采集设施）

| 资源点 | 自然形态 | 产出 | 采集设施 | 词条 |
|--------|----------|------|----------|------|
| [村镇](./村镇.md) | 小型聚落 | 基础人口 | [征兵办](./征兵办.md) | 村镇 · 征兵办 |
| [矿藏](./矿藏.md) | 裸露金属结构 | 金属 | [矿区](./矿区.md) | 矿藏 · 矿区 |
| [果地](./果地.md) | 野生果林 | 食物 | [果园](./果园.md) | 果地 · 果园 |
| [遗迹](./遗迹.md) | 前文明遗迹（**不可重建**） | 能源 | [能源站](./能源站.md) | 遗迹 · 能源站 |

## 建筑层（城区）

建筑层存储的就是**城区**（见 [城市模块化 · 城区定义](../../03-图层与地点/建筑层/README.md)）。

| 类型 | 说明 | 词条 |
|------|------|------|
| [废墟](./废墟.md) | **城区**（**状态：废墟**）；不可承载人口；须修复为**正常** | [废墟](./废墟.md) |
| [城市-外部](./城市-外部.md) | 独立外部城市（多格城区） | [城市-外部](./城市-外部.md) |

> **遗迹**（资源层、前文明、不可重建）与 **废墟**（建筑层、**状态为废墟的城区**）不是同一类存在。  
> [铁巢废墟](../../../04-设定/03-地点与场景/铁巢废墟.md) 为命名地区，不是上表网格类型。
