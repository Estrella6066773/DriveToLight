# 征兵办

## 摘要

建于 **[村镇](./村镇.md)** 资源点上的 **采集设施**（**唯一**玩法入口），将储量中的 **无归属**人口转化为可招募人口（产出细则 **待定**）。

## 玩家可见

简易征募处、登记台与驻点标识；由 [工程队](../../06-单位与交战/队伍系统.md) 建造与运维。

## 玩法关联

- 规则见 [荒野地点 · 征兵办](../荒野地点.md#资源点与采集设施)、[村镇](./村镇.md)（**已定**：仅资源点，**无**贸易 / 领袖页交互）。
- 资源点：[村镇](./村镇.md)；储量人口 **无归属**，提取前不计势力粮食需求。
