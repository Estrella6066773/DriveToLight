# 征兵办

## 摘要

建于 **[村镇](./村镇.md)** 资源点上的 **采集设施**（**唯一**玩法入口），将储量中的 **无归属**人口经工作提取为可安置 / 可编组人口。

## 玩家可见

简易征募处、登记台与驻点标识；由 [工程队](../../06-单位与交战/队伍系统.md) 建造与运维。

## 提取产量（已定 · 对齐住宅 50）

| 参数 | 首版默认 | 说明 |
|------|----------|------|
| 工作类型 | `extract_reserve` | 对征兵办实例下达；**不**在建造完成时自动产出 |
| `required_work_amount` | **30** | 满编工程队专精 **3 回合** |
| `output_per_work_completion` | **15** 人 | 无归属 → 写入目标容器（见下） |
| 折合周产（专精） | 约 **35** 人/办/周 | 7÷3 × 15 |
| 建造 | **30** 金属；对照 **3** 回合 | 见 [产出锚](../四种核心资源.md#四类资源产出锚已定) |

**去向（完成时择一，已定框架）**

1. **入城安置**：须目标城区有空余 [居民承载](../../03-图层与地点/建筑层/城区总览.md#居民承载)；提取后计粮食。
2. **随行人员**：写入相邻 / 指定队伍载荷随行池（须来源处预先规划住宅规则见 sy-14 / D-32）。
3. **设施闲置人口**：暂留 `idle_headcount`，供相邻格 [补员](../人口与迁移.md#补员--相邻格人员来源已定)。

提取前村镇储量**不计**势力粮食；提取后由接收城市的**城市领袖**纳入管辖。

**数量感（例 · 扩向 4 满编区）**：缺口若约 **100** 人，**2** 座征兵办专精约 **3** 周可补齐（未计战斗减员）。

## 村镇储量

见 [村镇](./村镇.md)：`reserve_min=60`，`reserve_max=120`（约 4～8 次满额提取）。
