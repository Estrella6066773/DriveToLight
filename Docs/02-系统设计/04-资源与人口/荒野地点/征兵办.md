# 征兵办

## 摘要

建于 **[村镇](./村镇.md)** 资源点上的 **采集设施**（**唯一**玩法入口），将储量中的 **无归属**人口转化为可招募人口（产出细则 **待定**）。

## 玩家可见

简易征募处、登记台与驻点标识；由 [工程队](../../06-单位与交战/队伍系统.md) 建造与运维。

## 玩法关联

- 规则见 [荒野地点 · 资源点与采集设施](../荒野地点.md#资源点与采集设施)、[村镇 · 资源点交互](./村镇.md#资源点交互已定)（**已定**：村镇**就是**资源点，**无**贸易 / 领袖页交互）。
- 资源点：[村镇](./村镇.md)；储量人口 **无归属**，提取前不计势力粮食需求。

## 程序数据（占位 · sy-23 / sy-14）

- 程序字段见 [recruitment-office-data](../../03-程序设计/03-数据字典/recruitment-office-data.md)；设计摘要见下表。**数值字段待定**。

| 字段 | 类型 | 说明 |
|------|------|------|
| `facility_id` | string | 固定 `recruitment_office`；见 `L5_facility_defs.csv` |
| `build_on_resource_type` | enum | **`village`**（仅可建于 [村镇](./村镇.md) 资源点） |
| `host_resource_point_id` | string | 绑定村镇格 / 储量实例 |
| `output_population_type_id` | string | **`pop_unaffiliated`**（无归属）；见 [城市管理系统 · 人力类型占位](../../04-资源与人口/城市管理系统.md#人力类型名单占位表占位--待策划定案--sy-23) |
| `work_type_id` | string | 提取工作类型（**待定**；候选 `extract_reserve` 或专用 `extract_population_reserve`） |
| `work_subject_kind` | enum | `facility_instance` 或 `resource_point`（**待定**） |
| `output_per_work_completion` | int | 单次工作完成产出人数（**待定**） |
| `max_extract_per_turn` | int | 每回合提取上限（**待定**） |
| `requires_engineer_team` | bool | 是否须 [工程队](../../06-单位与交战/队伍系统.md) 建造 / 运维（**待定**） |

- 提取后人口进入城区驻留、队伍**随行人员**，或暂留设施格 **`idle_headcount`**（供相邻格 [队伍补员](../人口与迁移.md#补员--相邻格人员来源已定)）；接纳入城时由接收城市的**城市领袖**纳入管辖。
