# 废墟

## 摘要

**建筑层**上**状态为废墟的城区**：损毁的城区实例，**不是**与「城区」并列的类型，**不是**资源层 [遗迹](./遗迹.md)（前文明遗迹、不可重建）。

## 玩家可见

曾有人居住或运转过的**城市残骸**；地图与信息层须让玩家**能认出**这是**废墟**，而非状态正常的城区或资源层遗迹。地图可**默认刷新**废墟，亦可在战斗 / 事件中将正常城区击毁为此状态。

**不可迁入**新居民；已在该城区的居民**仅可迁出**。**无法工作**——**特殊城区**模块工作区**不可启用**；**一般城区**设施工作区**停止运行**（**保留占格**）；见 [废墟失能](../../03-图层与地点/建筑层/城区总览.md#废墟失能)。须花费资源并完成修复，将**结构完整度**恢复至 **20% 及以上**且状态变为**正常**后，才**视为**正常城区（修复细则 **待定**）。

## 玩法关联

- 规则见 [城市模块化 · 废墟](../../03-图层与地点/建筑层/城区总览.md#废墟)、[废墟失能](../../03-图层与地点/建筑层/城区总览.md#废墟失能)、[荒野地点 · 遗迹与废墟](../荒野地点.md#遗迹与废墟术语强制区分)。
- **敌对占领**：外部城市**打为废墟**后，玩家**队伍占格**可完成占领（框架见 [废墟占领](../../05-城市与领袖/领袖与势力.md#废墟占领敌对--已定框架)）；城市整体失能条件 **待定**（sy-31）。
- 命名地区 [铁巢废墟](../../../04-设定/03-地点与场景/铁巢废墟.md) **不是** 本词条（网格城区状态）。
