# 城市-外部

## 摘要

荒野 **建筑层**上的**独立外部城市**：城区与玩家**同一类**（见 [势力系统 · 外部城市构成](../../05-城市与领袖/势力系统.md#外部城市构成)）；**不可整体移动**；每城 **1 名城市领袖**为**领袖关系**主体。

完整规则见 [势力系统](../../05-城市与领袖/势力系统.md)；本文为类型词条与玩家可见摘要。

## 玩家可见

- 规模**大于村镇**，有独立势力标识与城市领袖；玩家与之**交互**（贸易、委托、冲突、领袖关系变化等），**不能**当作矿藏 / 果地类资源点开采。
- 地图上表现为**多格占格**的固定据点；含 **1 个首都特殊城区**（势力主城区）与若干**一般城区**。
- 重要设定特例：[太阳城（骄阳之城）](../../../04-设定/03-地点与场景/太阳城.md)。

## 与村镇、移动城市的区别

| 对比 | **村镇** | **城市-外部** | **玩家移动城市** |
|------|----------|---------------|------------------|
| **图层** | 资源层资源点 | 建筑层城区集合 | 建筑层城区集合 |
| **交互** | **资源点流程**（揭示 / 建征兵办 / 工作转化；见 [村镇 · 资源点交互](./村镇.md#资源点交互已定)） | 贸易、委托、关系等（**非**资源点开采） | 玩家城市管理 |
| **移动** | 不可移动 | **不可**整体移动 | **停泊 / 航行** |
| **城区数量** | 不适用 | 与玩家**同一城区类**；整城不可移动；**无**连接 / 分离 / 迁移城区（[移动城市专属能力](../../05-城市与领袖/势力系统.md#移动城市专属能力已定)） | 可连接 / 分离、迁移城区（仅玩家移动城市，sy-15） |

## 机制要点（已定）

### 城区与占格

- 由一格或多格**城区**组成，挂载于 [建筑层](../../03-图层与地点/地图图层.md#建筑层与城区)；**与玩家城区同一类**。
- 与玩家移动城市**共用**城区运行时；**连接 / 分离 / 迁移城区 / 停泊航行**为 [移动城市专属能力](../../05-城市与领袖/势力系统.md#移动城市专属能力已定)，外部城与 AI **均不可**；其余能力由 [city-capability-flags](../../../03-程序设计/03-数据字典/city-capability-flags.md) gate 收敛。**占领**与招募 / 效忠 **互斥**（见 [占领、接管与招募](../../05-城市与领袖/领袖与势力.md#占领接管与招募分轨)）。
- **首都**：**仅 1 个**特殊城区（势力主城区）；城市领袖默认任职于此；其余为**一般城区**。
- 城区可进入**废墟**状态（与 [废墟](./废墟.md) 同一套规则）：**设施**与**特殊城区能力** [失能](../../03-图层与地点/建筑层/城区总览.md#废墟失能)。敌对城**打至废墟**后玩家**队伍占格**可占领（见 [废墟占领](../../05-城市与领袖/领袖与势力.md#废墟占领敌对--已定框架)）。城市整体失能 / 消亡细则 **待定**（sy-31）。

### 程序口径

- 与玩家移动城市**共用同一套城市与单位运行时**（城区、设施、资源、队伍、行动表等）；差异由 [city-capability-flags](../../../03-程序设计/03-数据字典/city-capability-flags.md) 与能力开关收敛，**不**为 NPC 单独裁减城市层实现（见 [势力系统 · 程序口径](../../05-城市与领袖/势力系统.md#外部城市程序口径已定)）。
- **城市层尽量完善**；**AI 决策层按需接入**——首版 AI 仅调用资源管理、生产单位、指挥单位三类接口（见 [外部城市 AI 模块](../../05-城市与领袖/势力系统.md#外部城市-ai-模块已定-首版)）；**招募成功后 AI 对该城停写**，由玩家接管指挥权。
- **未招募 / 已脱离**：不向玩家开放 [城市管理系统](../城市管理系统.md)；内部人口与劳动力由 [城市领袖](../../05-城市与领袖/领袖与势力.md#非玩家城市与人口调度) **抽象统管**。
- **招募 · 未效忠 / 效忠**：玩家**全部指挥权**；城市管理 **同一 UI**，未效忠 gate **禁用 + 说明**（见 [城市管理系统 · 招募 · 未效忠 UI](../城市管理系统.md#招募--未效忠-ui已定)）。

### 回合行动

- 每回合**玩家及其单位先于**所有外部城市行动；外部城市在 **AI 行动**阶段按 `game_seed` **伪随机顺序**各行动一次（见 [回合与行动表 · 外部城市行动顺序](../../07-玩法循环/回合与行动表.md#外部城市行动顺序)）。

### 关系与势力

- 每位**城市领袖**一套**领袖关系**，作用于**玩家方城市领袖**与**邻近城市的城市领袖**（邻近判定 sy-27）。
- 首版**领袖关系**不独立演化：AI **不**主动改写领袖关系表；**恶化**由人口损失驱动，**改善**由领袖委托与剧情事件驱动（见 [势力系统 · 领袖关系系统](../../05-城市与领袖/势力系统.md#领袖关系系统)）。
- 每位**城市领袖**始终展示 **3 条**委托供选；消失即补，且至少含 **2 种**类型；**已接取失败**扣完成奖励 **1/4** 领袖关系；**未接取**且已失效者**直接更换**（见 [领袖与势力 · 领袖委托](../../05-城市与领袖/领袖与势力.md#领袖委托已定-首版)）。
- 可归属同一**组织**；玩家与组织内一**城市领袖**发生直接领袖关系变化时，同组织其他领袖受**间接传导**（×1/4）；传导范围 sy-29。
- 关系档位与**未效忠 / 效忠**见 [势力系统 · 关系档位](../../05-城市与领袖/势力系统.md#领袖关系增减与演化已定-首版)；贸易基准价 **待定**（sy-28）。
