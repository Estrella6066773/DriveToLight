## 目标

定义城市之外荒野区域可刷新的地点类型及其产出，为探索与扩张提供目标。

## 范围

- **包含**：地点类型、产出资源、开采限制。
- **不包含**：刷新频率、勘探规则、具体产出数值。

## 详细说明

城市之外的地区视为[荒地](../../04-设定/03-地点与场景/荒地.md)，荒野格子上会刷新各类地点：

| 地点 | 产出 | 详细文档 |
|------|------|----------|
| [矿区](../../04-设定/03-地点与场景/矿区.md) | 建材 / 燃料 | [矿区](../../04-设定/03-地点与场景/矿区.md) |
| [果地](../../04-设定/03-地点与场景/果地.md) | 食物 | [果地](../../04-设定/03-地点与场景/果地.md) |
| [废墟](../../04-设定/03-地点与场景/废墟.md) | 建材 / 燃料 / 食物 | [废墟](../../04-设定/03-地点与场景/废墟.md) |
| [村镇](../../04-设定/03-地点与场景/村镇.md) | 食物 / 人口 | [村镇](../../04-设定/03-地点与场景/村镇.md) |
| [城市-外部](../../04-设定/03-地点与场景/城市-外部.md) | 建材 / 燃料 / 食物 / 人口 | [城市-外部](../../04-设定/03-地点与场景/城市-外部.md)（重要特例：[太阳城](../../04-设定/03-地点与场景/太阳城.md)） |

默认情况下，村镇和[城市-外部](../../04-设定/03-地点与场景/城市-外部.md)不能被[工程队](../01-核心系统/队伍系统.md)开采。
