## 目标

定义城市之外荒野区域可刷新的地点类型及其产出，为探索与扩张提供目标。

## 范围

- **包含**：地点类型、产出资源、开采限制。
- **不包含**：刷新频率、勘探规则、具体产出数值。

## 详细说明

城市之外的地区视为荒野，荒野格子上会刷新各类地点：

| 地点 | 产出 |
|------|------|
| 矿区 | 建材 / 燃料 |
| 果地 | 食物 |
| 废墟 | 建材 / 燃料 / 食物 |
| 村镇 | 食物 / 人口 |
| [城市](../01-核心系统/外部城市与组织关系.md) | 建材 / 燃料 / 食物 / 人口（外部城市，非移动城市） |

默认情况下，村镇和城市不能被[工程队](../01-核心系统/队伍系统.md)开采。
