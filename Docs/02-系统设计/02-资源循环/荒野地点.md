## 目标

定义城市之外荒野区域可刷新的地点类型及其产出，为探索与扩张提供目标。

## 范围

- **包含**：地点类型、产出资源、开采限制。
- **不包含**：刷新频率、勘探规则、具体产出数值。

## 详细说明

城市之外的地区视为[荒地](../../04-设定/03-地点与场景/荒地.md)，荒野格子上会刷新各类地点。

### 资源点揭示机制

荒野地点作为**资源点**，遵循 [单位类型与视野](../01-核心系统/单位类型与视野.md) 定义的揭示等级系统：

| 揭示等级 | 揭示内容 | 需要单位 |
|----------|----------|----------|
| **不揭示** | 仅知道此处有资源点（迷雾中的标记） | 无 |
| **揭示种类** | 知道是什么类型的地点（矿区/果地/废墟等） | [侦察队](../01-核心系统/单位类型与视野.md#1-侦察队) |
| **揭示储量** | 知道具体有多少资源可供采集 | [勘探队](../01-核心系统/单位类型与视野.md#2-勘探队) |

**探索流程**：
1. 派出[侦察队](../01-核心系统/单位类型与视野.md#1-侦察队)快速探索大范围区域，揭示资源点种类。
2. 针对有价值的资源点，派出[勘探队](../01-核心系统/单位类型与视野.md#2-勘探队)揭示具体储量。
3. 根据储量评估，决定是否派出[运输队](../01-核心系统/单位类型与视野.md#3-运输队)和[工程队](../01-核心系统/单位类型与视野.md#4-工程队)进行采集/建设。

### 荒野地点类型

| 地点 | 产出 | 详细文档 |
|------|------|----------|
| [矿区](../../04-设定/03-地点与场景/荒野地块/矿区.md) | 建材 / 燃料 | [矿区](../../04-设定/03-地点与场景/荒野地块/矿区.md) |
| [果地](../../04-设定/03-地点与场景/荒野地块/果地.md) | 食物 | [果地](../../04-设定/03-地点与场景/荒野地块/果地.md) |
| [废墟](../../04-设定/03-地点与场景/荒野地块/废墟.md) | 建材 / 燃料 / 食物 | [废墟](../../04-设定/03-地点与场景/荒野地块/废墟.md) |
| [村镇](../../04-设定/03-地点与场景/荒野地块/村镇.md) | 食物 / 人口 | [村镇](../../04-设定/03-地点与场景/荒野地块/村镇.md) |
| [城市-外部](../../04-设定/03-地点与场景/荒野地块/城市-外部.md) | 建材 / 燃料 / 食物 / 人口 | [城市-外部](../../04-设定/03-地点与场景/荒野地块/城市-外部.md)（重要特例：[太阳城](../../04-设定/03-地点与场景/太阳城.md)） |

默认情况下，村镇和[城市-外部](../../04-设定/03-地点与场景/荒野地块/城市-外部.md)不能被[工程队](../01-核心系统/队伍系统.md)开采。
