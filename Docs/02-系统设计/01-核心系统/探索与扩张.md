## 目标

定义城市停下后如何与外部世界互动，包括勘探、建立设施、补充资源与人口。

## 范围

- **包含**：停泊后的勘探、资源点确认、建立采集站 / 驿站、运输补给、前往外部据点。
- **不包含**：设施建造消耗公式、据点生成规则、界面交互细节、完整数值平衡。

## 详细说明

### 系统定位

探索与扩张承接 [地图与移动](./地图与移动.md) 中的**停泊**状态：移动城市停下后，玩家通过队伍把城市能力带到周边荒野，换取资源、信息、路线选择与外部据点关系。它是「停下交互 → 勘探与开发 → 资源与人口管理 → 再次移动」的中段。

### 基本流程

```mermaid
flowchart LR
    A[城市停泊] --> B[派出队伍]
    B --> C[揭示地点与资源]
    C --> D{是否值得开发?}
    D -->|是| E[建设设施]
    D -->|否| F[记录信息并调整路线]
    E --> G[运输与维护]
    G --> H[补给主城]
    F --> I[准备再次移动]
    H --> I
```

| 阶段 | 玩家在做什么 | 主要系统 | 产出 |
|------|--------------|----------|------|
| 停泊准备 | 确认主城已停下，规划外出目标 | [地图与移动](./地图与移动.md)、[回合与行动表](../02-玩法循环/回合与行动表.md) | 可执行停下后的交互 |
| 派出队伍 | 从主城派出侦察队、勘探队、运输队或工程队 | [队伍系统](./队伍系统.md)、[单位类型与视野](./单位类型与视野.md) | 外出单位与指令表 |
| 揭示信息 | 发现资源点、地形、危险或据点 | [单位类型与视野](./单位类型与视野.md)、[通讯与飞讯系统](./通讯与飞讯系统.md) | 已同步或待同步的信息 |
| 判断开发 | 依据储量、距离、风险与当前资源决定是否开发 | [荒野地点](../02-资源循环/荒野地点.md)、[四种核心资源](../02-资源循环/四种核心资源.md) | 开发、放弃或延后 |
| 建设设施 | 建立采集站、驿站、桥梁等荒野设施 | [地图图层](./地图图层.md)、[队伍系统](./队伍系统.md) | 设施与后续维护需求 |
| 运输补给 | 将资源点产出或据点补给带回主城 | [队伍系统](./队伍系统.md)、[四种核心资源](../02-资源循环/四种核心资源.md) | 资源回流与行动范围扩大 |

### 勘探与信息同步

- 侦察队优先揭示资源点**种类**，勘探队进一步揭示资源点**储量**；揭示等级见 [单位类型与视野 · 资源点揭示等级](./单位类型与视野.md#资源点揭示等级)。
- 外出单位看到的信息不一定立即进入玩家视野；同步规则见 [通讯与飞讯系统](./通讯与飞讯系统.md)。
- 若信息尚未同步，玩家可以基于旧信息继续下达指令，但要承担路径、风险或资源判断失准的后果。

### 设施建设

- 工程队负责在合法位置建设荒野设施；设施合法性由 [地图图层](./地图图层.md) 的地形、资源层、设施层影响规则共同决定。
- 典型设施包括采集站、驿站、桥梁等：
  - **采集站**：围绕资源点建立，用于后续采集和运输。
  - **驿站**：为运输、补给或路线中继服务。
  - **桥梁**：用于跨越裂隙等地形障碍，具体条件见 [地图图层 · 影响规则示例](./地图图层.md#影响规则示例)。
- 设施建设是按回合推进的工作；工程队工作规则见 [单位类型与视野 · 工程队模板](./单位类型与视野.md#4-工程队模板)。

### 外部据点交互

- 村镇与外部城市属于 [荒野地点](../02-资源循环/荒野地点.md) 中的固定据点，可提供人口、资源、贸易、任务或关系变化。
- 外部城市不是玩家的 [移动城市](./地图与移动.md)，不能被工程队开采。
- 与外部城市的关系变化归入 [外部城市与组织关系](./外部城市与组织关系.md)；同组织内的间接关系变化不在本文重复展开。

### 与再次移动的关系

探索与扩张的结果最终回到 [核心循环](../02-玩法循环/核心循环.md)：

| 结果 | 对下一次移动的影响 |
|------|--------------------|
| 资源充足 | 支撑燃料、食物、建材与人口消耗，扩大可行路线 |
| 路线信息更完整 | 降低撞上障碍、危险或无效资源点的风险 |
| 建立中继设施 | 延长运输和补给能力，允许主城更早离开当前位置 |
| 关系改善 | 提供贸易、支援或章节任务入口 |
| 开发失败或信息延迟 | 可能迫使玩家停留更久、改线或牺牲城区模块 |
