## 建议完善顺序

1. **OPEN-023 ~ OPEN-026**（未知死亡）→ 阻塞指令/通讯/UI 完整落地。
2. **OPEN-013 通讯** → 与 OPEN-025 一并讨论死亡宣告通道。
3. **OPEN-014 揭示** → 可定 `vision_sync_pending` 过期与合并规则。
4. **OPEN-008 队伍数值** → 可填各模板 SO 默认参数（配置模板搭建的前置工作）。
5. **04-数值框架** → 战斗减员、建筑损伤、设施耐久、资源公式（**在配置模板搭建完成后**再开）。
6. 地图 / 城市 / 太阳 → 按 [数据字典 README](03-数据字典/README.md) 逐张补表。
7. **OPEN-037、OPEN-039 ~ OPEN-040**（旋转吸附、快捷键、渲染管线）→ 可在 Input / Map 表现层原型前并行讨论。
8. **OPEN-031 / OPEN-048** → 后续新增城模块或城区能力时，统一补 `GameplayEffect` / `GameplayAbility` 配置。
