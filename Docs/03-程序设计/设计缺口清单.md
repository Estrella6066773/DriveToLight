1. **sy-25 通讯站主动消耗** → 资源数值。

5. **04-数值框架** → 战斗减员、建筑损伤、设施耐久、资源公式（**在配置模板搭建完成后**再开）。
6. 地图 / 城市 / 太阳 → 按 [数据字典 README](03-数据字典/README.md) 逐张补表。
7. **sy-17、sy-18 ~ sf-01**（旋转吸附、快捷键、渲染管线）→ 可在 Input / Map 表现层原型前并行讨论。
8. **sy-12 / sy-25** → 后续新增城模块或城区能力时，统一补 `GameplayEffect` / `GameplayAbility` 配置。
