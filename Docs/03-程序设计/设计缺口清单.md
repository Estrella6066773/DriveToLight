1. **OPEN-023 ~ OPEN-026**（未知死亡）→ 阻塞指令/通讯/UI 完整落地。

5. **04-数值框架** → 战斗减员、士气、资源公式。
6. 地图 / 城市 / 太阳 → 按 [数据字典 README](03-数据字典/README.md) 逐张补表。
