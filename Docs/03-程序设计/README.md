# 程序设计

本目录存放《循光之城》的**运行时架构、逻辑规范、数据字典、枚举、引擎映射**。

## 子目录

| 目录 | 用途 |
|------|------|
| [01-架构总览/](01-架构总览/) | 模块划分、依赖关系、启动流程 |
| [02-运行时逻辑/](02-运行时逻辑/) | 各模块的接口、判定顺序、状态机 |
| [03-数据字典/](03-数据字典/) | 数据表字段定义、关系、示例（当前暂无实现数据表，见 [README](03-数据字典/README.md)） |
| [04-引擎映射/](04-引擎映射/) | 代码与 Unity 资源/场景的对应关系 |
