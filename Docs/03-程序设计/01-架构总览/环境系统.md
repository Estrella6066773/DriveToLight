## 运行时数据模型

### SunMotionState

太阳运动运行时状态（持有字段）：

| 字段 | 类型 | 说明 |
|------|------|------|
| `initialVelocityV0` | float | 初速度 v₀ |
| `accelerationA` | float | 恒定加速度 a |
| `turnIndex` | int | 环境结算计数 t |
| `cellAccumulator` | float | 格残差 accum |
| `motionEnabled` | bool | 是否启用太阳运动（照射区完全离图后为 false） |
| `currentLightFrontierRow` | int | 当前光带前沿海拔行（格坐标，具体映射待定） |

### LightBandType 枚举

```
Daylight   = 0  // 日光带
Twilight   = 1  // 黄昏带
DarkZone   = 2  // 暗渊带（太阳移动启用时）
GlobalDark = 3  // 全图暗渊（sun_motion_enabled=false）
```

### EnvironmentEffectState

单格环境层效果聚合：

| 字段 | 类型 | 说明 |
|------|------|------|
| `cellCoord` | HexCoord | 所属格坐标 |
| `groundSurface` | GroundSurfaceState? | 地面表层效果（可为 null） |
| `activeWeather` | WeatherRegionState? | 该格所属天气区域（可为 null） |
| `cityTrace` | CityTraceState? | 城市轨迹效果（可为 null） |

### GroundSurfaceState

| 字段 | 类型 | 说明 |
|------|------|------|
| `surfaceId` | string | 对应 L2 CSV 的 `environment_id`（woodland / scorched_earth 等） |
| `remainingTurns` | int | 剩余持续回合（-1 表示无限） |
| `appliedTurn` | int | 产生时的回合编号 |

### WeatherRegionState

| 字段 | 类型 | 说明 |
|------|------|------|
| `weatherId` | string | sandstorm / heavy_rain / fog |
| `centerCoord` | HexCoord | 区域中心 |
| `radiusCells` | int | 影响半径（格数） |
| `remainingTurns` | int | 剩余持续回合 |
| `nextRefreshTurn` | int | 下次刷新回合编号 |

### CrossModifierEntry

跨图层修正运行时条目：

| 字段 | 类型 | 说明 |
|------|------|------|
| `priority` | int | 优先级（越小越先结算） |
| `whenLayer` | string | 触发条件·图层（terrain / environment / facility） |
| `whenIds` | string[] | 触发条件·类型 ID |
| `requireLayer` | string | 修正目标·图层 |
| `requireIds` | string[] | 修正目标·类型 ID |
| `statKey` | string | 修正的属性键 |
| `op` | ModifierOp | Add / Multiply / Set |
| `value` | float | 修正值 |

## 服务接口

### SunMotionService

```csharp
namespace Yanxu.Environment.Sun
{
    public class SunMotionService
    {
        // 从 ChapterEnvironmentProfileSO 加载运动参数
        void LoadProfile(ChapterEnvironmentProfileSO profile);

        // 每环境回合调用一次：计算 v(t)、推移光带、检测停用
        void TickSunMotion();

        // 查询指定格子当前光带状态
        LightBandType GetLightBand(HexCoord coord);

        // 查询太阳运动是否已停用（全图暗渊）
        bool IsMotionEnabled { get; }
    }
}
```

### EnvironmentEffectService

```csharp
namespace Yanxu.Environment.Effects
{
    public class EnvironmentEffectService
    {
        // 每环境回合调用：到期检测与清理
        void TickEnvironmentEffects();

        // 查询指定格子的环境效果聚合
        EnvironmentEffectState GetEffectState(HexCoord coord);

        // 添加地面表层效果（由单位行为触发）
        void ApplyGroundSurface(HexCoord coord, string surfaceId, int durationTurns);

        // 移除指定格子的地面表层效果
        void RemoveGroundSurface(HexCoord coord);

        // 添加城市轨迹
        void ApplyCityTrace(IEnumerable<HexCoord> path, int durationTurns);
    }
}
```

### WeatherService

```csharp
namespace Yanxu.Environment.Effects
{

    {
        // 每环境回合调用：区域刷新或消退

}
```

### EnvironmentOrchestrator

编排器，绑定于 [环境行动阶段](../../02-系统设计/07-玩法循环/回合与行动表.md#回合阶段)：

```csharp
namespace Yanxu.Environment
{
    public class EnvironmentOrchestrator
    {
        // 每回合环境行动阶段调用，按顺序执行：
        //   Step1: sunMotion.TickSunMotion()
        //   Step2: effectService.TickEnvironmentEffects()
        //         + weatherService.TickWeather()
        //   Step3: crossModifierPipeline.ResolveAll()
        void RunEnvironmentPhase();

        // 章节切换
        void OnChapterSwitch(int chapterId, ChapterEnvironmentProfileSO profile);
    }
}
```

## GAS-lite 集成方式

环境效果通过 GAS-lite 挂载于 **格子 ASC**（与 [地图图层 · 地形类型清单](../../02-系统设计/03-图层与地点/地图图层.md#地形类型清单) 中「每格地形挂载 ASC」一致）：

```
单格 ASC（HexCell.AbilitySystemComponent）：
  ├── 地形 GE（永久：平原/丘陵/山地等的 GameplayEffect 实例）
  ├── 环境 GE（有期限：树林、焦土、天气等）
  └── 跨图层修正值（cross_modifiers 写入 Attribute）
```

环境 GE 使用 `DurationType = HasDurationTurn`，到期由 `AbilitySystemComponent` 自动移除；跨图层修正走 `ModifierOp` 写入对应格子的 `GameplayAttribute`。

## 配置管线

| 数据来源 | 运行时加载方式 | 对应类别 |
|----------|---------------|----------|
| `L1_terrain_defs.csv` | 网格生成时→格子 ASC 挂载永久 GE | 地形 |
| `L2_environment_defs.csv` | 环境效果生成时→格子 ASC 挂载有期限 GE | 环境 |
| `L0_cross_modifiers.csv` | CrossModifierPipeline 加载 | 跨图层修正 |
| `ChapterEnvironmentProfileSO` | ChapterEnvironmentService 加载 | 章节配置 |
| `EnvironmentConfigSO` | EnvironmentOrchestrator 初始化 | 全局配置 |

## 与现有模块的衔接

### 与 Yanxu.Turn 的关系

- `EnvironmentOrchestrator.RunEnvironmentPhase()` 由 `TurnPhaseMachine` 在 `EnvironmentAction` 阶段调用
- 现有 `ResponseCheckPipeline` 保持不变；环境新检测器通过实现 `IResponseDetector` 注册

### 与 Yanxu.Map 的关系

- 格子 ASC 创建在 `HexGridStore.AddCell()` 中；环境系统通过 `IGridQuery<HexCoord>` 访问网格
- 光带标记不修改 `HexCell.IsBlocked`，走独立查询通道

### 与 Yanxu.City 的关系

- 航行轨迹由 `MobileCityStateService` 在航行移动完成后写入坐标；环境系统据此生成 `CityTraceState`
- 停泊状态不受光带推移影响（停泊时仍占世界格）
