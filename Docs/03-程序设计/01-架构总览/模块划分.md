| Resource | 金属 / 食物 / 能源 / 人口四类资源 | | `Yanxu.Resource` |

| Audio | 音频管理 | | `Yanxu.Audio` |

## 职责边界

- **运行时职责**：MVC 仅用于运行时逻辑组织与交互，不承担文档追踪职责。
- **文档职责**：文档追踪由文档流程维护，统一记录在 [待细化追踪](../../00-规范/待细化追踪.md) 与各文档的「待确认事项」中。
