## 目标

为六边形格地图**每一图层**提供与 [地形 profile](./tables/terrain_profile_config.csv) 同级的 **type + profile** CSV，使数值策划**改表即可定案**，不必逐条口述或在文档表格与程序常量之间来回抄数。

**原则**：

| 原则 | 说明 |
|------|------|
| **一行一类** | `*_profile_config.csv`：该类型全部可调玩法列 |
| **速查表** | `*_type_config.csv`：枚举 + 关键列，与 profile 对齐 |
| **枚举独立** | 三档通行性、城区类别等 → `*_enum.csv` |
| **实现映射** | GE/GA 目录仅程序实现；**策划主读 profile** |
| **设施特例** | 地形类设施的宿主格覆盖 → [facility_terrain_class_config.csv](./tables/facility_terrain_class_config.csv) |

**单位层**不列入本体系：单位在格上移动，不以「格基线类型」配置；见 [队伍与战斗数据结构](./队伍与战斗数据结构.md)。

## 统一表型（每图层）

```
{layer}_type_config.csv      → 枚举、显示名、profile_id、关键列速查
{layer}_profile_config.csv   → 策划主表（权威数值）
{layer}_*_enum.csv           → 该层专用枚举（可选）
{layer}_*_catalog.csv        → 实现映射 / 白名单目录（可选）
```

## L1 地形层

已落地。详见 [地形类型配置数据结构](./地形类型配置数据结构.md)。

附属：`facility_terrain_class_config`（桥梁/隧道有效时覆盖）、`facility_build_allow_catalog`（建造白名单 → GE）。

---

## L2 环境层

**职责**：格上表层状况（树林、焦土、弹坑）与区域/大气效果（沙暴、暴雨、迷雾、移动城市轨迹）。与地形**正交**，各类型自有堆叠规则。

### `environment_type_config`（速查）

| 字段 | 说明 |
|------|------|
| `environment_type_id` | PK |
| `profile_id` | FK |
| `environment_subtype` | `ground_surface` / `weather` / `city_trace` |
| `stack_rule` | `exclusive` / `stackable` / `regional` |
| `config_status` | `defined` / `pending` |

### `environment_profile_config`（主表）

| 字段 | 说明 |
|------|------|
| `path_cost_delta` | 对单位路径等效的加减（空=0） |
| `unit_enter_allowed_override` | 覆盖地形基线（空=不覆盖） |
| `vision_radius_modifier_pct` | 单位视野修正 %（空=0） |
| `watchtower_vision_modifier_pct` | 哨塔视野修正 %（密林为负） |
| `default_duration_turns` | 默认存续回合；空=直至移除 |
| `facility_build_mode_override` | 覆盖建造档位（空=不覆盖） |

枚举：[environment_subtype_enum.csv](./tables/environment_subtype_enum.csv)、[environment_stack_rule_enum.csv](./tables/environment_stack_rule_enum.csv)

---

## L3 资源层

**职责**：资源点类型、储量、揭示、采集设施白名单。与地形正交。

### `resource_point_profile_config`（主表）

| 字段 | 说明 |
|------|------|
| `reserve_resource_kind` | `metal` / `food` / `energy` / `population` |
| `facility_build_allow_type_ids` | 白名单；优先级高于地形档位 |
| `point_rebuildable` | 资源点本体可否重建（遗迹 **false**） |
| `default_reveal_level` | `hidden` / `kind` / `amount` |
| `reserve_amount_min` / `max` | 刷新储量区间（**待定**填数） |
| `extract_work_type_id` | 开采工作类型 |

`resource_point_facility_allow.csv` 保留为白名单 → GE 映射；**数值以 profile 为准**。

---

## L4 建筑层（城区）

**职责**：建筑层 = **城区**。每格至多一个城区实例；类别 × 状态正交。

### `district_profile_config`（主表）

| 字段 | 说明 |
|------|------|
| `district_category` | `special` / `general` |
| `base_resident_cap` | 基础居民承载 |
| `move_grids_per_load_step` | 负载成本：每 x 格 |
| `repair_resource_cost` | 负载成本：累积 y 点修复代价 |
| `repair_resource_kind` | 修复资源类型（默认 `metal`） |
| `facility_slot_count` | 一般城区设施建造位数 |
| `ruins_integrity_threshold_pct` | 低于此完整度转废墟（默认 **20**） |
| `allows_work_when_ruins` | 废墟是否可工作（默认 **false**） |

枚举：[district_category_enum.csv](./tables/district_category_enum.csv)、[district_state_enum.csv](./tables/district_state_enum.csv)

---

## L5 设施层

**职责**：占格类 / 辅助类设施类型参数。地形类宿主格覆盖见 `facility_terrain_class_config`（**设施基本能力**，非 Buff 目录）。

### `facility_type_profile_config`（主表）

| 字段 | 说明 |
|------|------|
| `facility_category` | `occupying` / `auxiliary` |
| `occupying_subtype` | `terrain` / `production` / `storage` / `dwelling` |
| `production_tier` | `reserve_to_resource` / `resource_to_asset` / 空 |
| `build_work_type_id` | 建造工作 |
| `build_turns` | 建造回合数 |
| `base_durability_max` | 耐久上限 |
| `maintain_work_type_id` | 运维工作 |
| `storage_cap_bonus` | 仓储类：容量加成 |
| `extra_resident_cap` | 屋舍：额外承载 |
| `population_loss_reduction_pct` | 城墙：本格人口损失减免 % |

---

## L6 物品层

**职责**：可落于格上或被拾取的物品类型；堆叠与搬运规则。

### `item_profile_config`（主表）

| 字段 | 说明 |
|------|------|
| `item_category` | `resource` / `asset` / `equipment` |
| `max_stack_per_hex` | 单格堆叠上限 |
| `max_stack_in_warehouse` | 仓库堆叠上限 |
| `weight_per_unit` | 搬运重量（**待定**） |
| `pickup_requires_team` | 是否须队伍拾取 |

---

## 图层间影响规则

跨层修正（哨塔+密林、地形+桥梁白名单等）列入 [layer_interaction_rule_config.csv](./tables/layer_interaction_rule_config.csv)，每行含 `priority`、触发图层/类型、效果列；**须填优先级**（见地图图层规范）。

---

## 程序映射（总览）

| 步骤 | 行为 |
|------|------|
| 地图生成 / 读档 | 各层 profile → hex 分层状态；地形/环境 → ASC GE |
| 移动 / 视野 / 建造 | 读**合成参数**：地形基线 + 环境修正 + 有效地形类设施 + 图层间规则 |
| 设施建成 / 击毁 | `FacilityTerrainClassService`；耐久与辅助类效果走设施 SO |
