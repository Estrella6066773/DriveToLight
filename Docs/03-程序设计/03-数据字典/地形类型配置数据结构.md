## 目标

地图 **地形层** 的玩法差异不写在程序 `if (terrain == X)` 中，而拆为可复用的 **GameplayEffect / GameplayAbility**，经 **地形档案（profile）** 装配到每格的 `AbilitySystemComponent`（ASC）。关卡策划调参时改 CSV → 导入 SO；新增地形类型时组合已有能力即可。

叙事与几何规则（裂谷桥梁直线、移动城市 footprint）仍写在 [地图图层](../../02-系统设计/03-图层与地点/地图图层.md)。

## 权威 CSV（三张表）

| 文件 | 职责 |
|------|------|
| [terrain_type_config.csv](./tables/terrain_type_config.csv) | 地形枚举 → `profile_id` |
| [terrain_profile_config.csv](./tables/terrain_profile_config.csv) | 每 profile 挂载的 GE / GA / Tag 列表（**SO 装配清单**） |
| [terrain_capability_catalog.csv](./tables/terrain_capability_catalog.csv) | 地形专属能力目录（须单独设计实现的 GE/GA） |

**改装配**：编辑 `terrain_profile_config.csv`。  
**改能力语义或数值**：先更新 `terrain_capability_catalog.csv`，再同步 Unity 中对应 `GameplayEffectConfigSO` / `GameplayAbilityConfigSO`。

## 表 1：`terrain_type_config`

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `terrain_type_id` | string | **PK** | `plain` / `hill` / `mountain` / `river` / `swamp` / `rift` |
| `display_name_zh` | string | 非空 | 显示名 |
| `profile_id` | string | FK 可空 | → `terrain_profile_config.profile_id` |
| `config_status` | string | `defined` / `pending` | `pending` 可不挂 profile |
| `notes_zh` | string | 可空 | 备注 |

## 表 2：`terrain_profile_config`

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `profile_id` | string | **PK** | 如 `terrain_profile_hill` |
| `terrain_type_id` | string | FK | 对应地形枚举 |
| `granted_tags_json` | string | 分号分隔 | 格 ASC 初始化授予的 Tag，如 `Terrain.Hill` |
| `gameplay_effect_ids` | string | 分号分隔 | 初始化 Apply 的 GE id 列表 |
| `gameplay_ability_ids` | string | 分号分隔 | 注册到格 ASC 的 GA id（含被动规则型） |
| `notes_zh` | string | 可空 | 备注 |

生成地图格时：读 `terrain_type_id` → `profile_id` → 在 hex ASC 上 Apply 所列 GE、注册 GA、授予 Tag。

## 表 3：`terrain_capability_catalog`

每条 **capability_id** 对应一个须实现 SO + executor（若为 GA）的玩法单元。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `capability_id` | string | **PK**；与 SO 资产 `effect_id` / `ability_id` 对齐 |
| `capability_kind` | string | `gameplay_effect` / `gameplay_ability` |
| `display_name_zh` | string | 策划可读名 |
| `executor_key` | string | GA 专用；程序入口（见 [Effect 与能力解析](../02-运行时逻辑/Effect与能力解析.md)） |
| `gameplay_effect_id` | string | GE 专用；自引用或 SO id |
| `attribute_id` | string | GE 修正的 Attribute，如 `terrain_path_cost_equiv` |
| `modifier_op` | string | `set` / `add` / `multiply` |
| `modifier_value` | number | 修正值 |
| `granted_tags_json` | string | GE/GA 授予 Tag |
| `blocked_tags_json` | string | GE/GA 阻断 Tag（如禁止某类建造） |
| `required_tags_json` | string | GA 激活所需 Tag |
| `rule_guard_zh` | string | 前置条件说明；程序侧写入 `rule_guard_json` |
| `impl_status` | string | `defined` / `pending` |
| `notes_zh` | string | 备注 |

### 首版能力一览

| capability_id | 类型 | 作用 |
|---------------|------|------|
| `GE_Terrain_BlockAllBuild` | GE | 沼泽等：阻断 `Capability.Build` |
| `GE_Terrain_PathCost_*` | GE | 路径等效格数（丘陵 **2**、沼泽 **3**、河流 **4**、裂谷 **1** 等） |
| `GE_Terrain_BlockBuild_*` | GE | 通过 `blocked_tags` 限制建造大类 |
| `GE_Terrain_BlockUnitEnter` | GE | 禁止一般单位进入（裂谷） |
| `GE_Terrain_PostTunnel` | GE | 隧道建成后覆盖路径消耗并解除辅助类封锁 |
| `GA_TerrainFlattenToPlain` | GA | 丘陵工程改造为平原 |
| `GA_TerrainBuildTunnel` | GA | 山地隧道（`TryBuild` + facility SO） |
| `GA_TerrainBuildBridge` | GA | 裂谷桥梁（`TryBuild` + 几何 guard） |
| `GA_TerrainRule_MobileCityCrossRift` | GA | 移动城市跨裂谷被动规则 |

### 新增 Tag（地形）

| Tag | 说明 |
|-----|------|
| `Terrain.Plain` / `Terrain.Hill` / `Terrain.Mountain` / `Terrain.River` / `Terrain.Rift` / `Terrain.Swamp` | 地形种类标记 |
| `Capability.Build.ProductionStorage` | 允许建生产/仓储占格类 |
| `Capability.Build.Auxiliary` | 允许建辅助类 |
| `Capability.Move.Enter` | 允许进入并停留该格 |
| `Blocked.Build.Auxiliary` | 辅助类建造被地形效果阻断 |
| `Actor.MobileCity` | 移动城市 actor（用于裂谷例外规则） |

## 程序映射

| 步骤 | 行为 |
|------|------|
| 地图生成 / 读档 | `terrain_type_id` → 查 profile → hex `AbilitySystemComponent` 初始化 |
| 移动结算 | 读 hex ASC `terrain_path_cost_equiv`（GE 贡献快照） |
| 建造 | `TryBuild` GA 校验 hex ASC 的 `blocked_tags` / `required_tags` |
| 地形改造 | `GA_TerrainFlattenToPlain` 完成后替换 `terrain_type_id` 并 **重载 profile** |
| 设施改变地形 | 隧道/桥梁建成后 Apply 或移除对应 GE（如 `GE_Terrain_PostTunnel`） |

| CSV 行 | Unity SO（规划） |
|--------|------------------|
| `terrain_profile_config` | `TerrainProfileConfigSO` |
| `terrain_capability_catalog` 中 GE | `GameplayEffectConfigSO` |
| `terrain_capability_catalog` 中 GA | `GameplayAbilityConfigSO` |
