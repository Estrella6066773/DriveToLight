## 表：`L1_terrain_defs.csv`

**表格作用说明**

地形层**唯一策划主表**。每行一种地形，描述该格在**无有效地形类设施**时的玩法基线：路径消耗、能否进入、建造档位、移动城市通行性，以及允许例外建造的设施白名单。完整说明见 [地图图层配置数据结构 · L1](./地图图层配置数据结构.md#l1_terrain_defscsv)。

**字段（策划列名 · 程序键）**

| 策划列名 | 程序键 | 类型 | 约束 | 说明 |
|----------|--------|------|------|------|
| **类型编号** | `terrain_id` | string | **PK** | 稳定英文 id |
| **显示名称** | `display_name_zh` | string | 非空 | 地形显示名 |
| **路径消耗（格）** | `path_cost_equiv` | int | ≥1 | 单位路径等效格数 |
| **单位可否进入** | `unit_enter_allowed` | bool | | 填 **是** / **否**；否时阻断 `Capability.Move.Enter` |
| **建造档位** | `build_mode` | string | enum:build_mode | 填显示名称：全部允许 / 仅辅助类 / 全部禁止 |
| **允许建造的设施** | `build_allow_facility_ids` | string | `;` 分隔 | 设施类型编号或中文名；优先于建造档位 |
| **移动城市通行** | `mobile_city_pass` | string | enum:mobile_city_pass | 可通过 / 可跨越 / 不可通过 |
| **移动城市路径消耗（格）** | `mobile_city_path_cost_equiv` | int | 可空 | 空 = 与路径消耗相同 |
| **改造能力说明** | `transform_ability_desc` | string | 可空 | 自然语言；详表见 [能力说明.csv](./tables/shared/能力说明.csv) |
| **程序标签** | `tags` | string | `;` 分隔 | 授予 Tag，如 `Terrain.Plain` |
| **状态** | `status` | string | enum:record_status | **已定** / **待定** |
| **备注** | `notes_zh` | string | 可空 | 备注 |

## 程序映射

| 步骤 | 行为 |
|------|------|
| 导入 | 中文列名经 `column_registry` 映射；枚举显示名经 `enums` 反查 `enum_key` |
| 地图生成 | 读地形行 → Apply `ge_catalog` 对应 GE → 授予 `tags` |
| 移动 | 读合成后 `path_cost_equiv` |
| 建造 | `TryBuild`：白名单 Tag → `build_mode` → `facility_defs.rule_guard` |
| 地形类设施 | `FacilityTerrainClassService` 按 L5 `on_valid_*` 覆盖；击毁回退 |
