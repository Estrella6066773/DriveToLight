## 表：`terrain_defs.csv`

**表格作用说明**

地形层**唯一策划主表**。每行一种地形，描述该格在**无有效地形类设施**时的玩法基线：路径消耗、能否进入、建造档位、移动城市通行性，以及允许例外建造的设施白名单。完整说明见 [地图图层配置数据结构 · L1](./地图图层配置数据结构.md#l1-terrain_defscsv)。

**字段**

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `terrain_id` | string | **PK** | 程序枚举 id |
| `display_name_zh` | string | 非空 | 显示名 |
| `path_cost_equiv` | int | ≥1 | 单位路径等效格数 |
| `unit_enter_allowed` | bool | | `false` 时阻断 `Capability.Move.Enter` |
| `build_mode` | string | enum:build_mode | 可建造设施基础档位 |
| `build_allow_facility_ids` | string | `;` 分隔 | 允许设施白名单（`facility_id`）；优先于 `build_mode` |
| `mobile_city_pass` | string | enum:mobile_city_pass | 移动城市通行性 |
| `mobile_city_path_cost_equiv` | int | 可空 | 空 = 与 `path_cost_equiv` 相同 |
| `transform_ability_id` | string | 可空 | 地形改造 GA（非设施）；空 = 无 |
| `tags` | string | `;` 分隔 | 授予 Tag，如 `Terrain.Plain` |
| `status` | string | enum:record_status | `defined` / `pending` |
| `notes_zh` | string | 可空 | 备注 |

### 关联（非本表字段）

| 主题 | 位置 |
|------|------|
| 白名单 → `Allow.Build.*` Tag | [_program/build_grant_tags.csv](./tables/_program/build_grant_tags.csv) |

| 资源点建造白名单 | [L3 resource_defs](./tables/L3_resource/resource_defs.csv) |
| GE 导入映射 | [_program/ge_catalog.csv](./tables/_program/ge_catalog.csv) |

### 枚举组（见 shared/enums.csv）

- **build_mode**：`all` / `auxiliary_only` / `none`
- **mobile_city_pass**：`passable` / `crossable` / `blocked`（`crossable` 由 `EvaluateMobileCityCrossableFootprint` 校验 footprint）

## 程序映射

| 步骤 | 行为 |
|------|------|
| 地图生成 | 读 `terrain_defs` 行 → Apply `ge_catalog` 对应 GE → 授予 `tags` |
| 移动 | 读合成后 `path_cost_equiv` |
| 建造 | `TryBuild`：白名单 Tag → `build_mode` → `facility_defs.rule_guard` |
| 地形类设施 | `FacilityTerrainClassService` 按 L5 `on_valid_*` 覆盖；击毁回退 |
