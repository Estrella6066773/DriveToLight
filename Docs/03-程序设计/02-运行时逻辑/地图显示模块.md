# 地图显示模块

管理并显示每个格子的地形等信息的初版显示模块。数据与显示分离：`Yanxu.Map` 负责格子数据，本模块（`Yanxu.MapView` 程序集）负责显示模型与场景表现。

## 目标与范围

- **包含**：每格显示信息的管理（显示模型）、按地图工具（DTLM 文件）色卡颜色区分地形的占位表现、与地图导入工具的联动刷新。
- **不包含**：正式渲染方案（待 sf-01 渲染管线选型）、L1～L5 配置表驱动的格状态合成（sf-13）、视野迷雾、城区与资源点的专项表现（见 [待确认事项](#待确认事项)）。

## 已定口径

| 主题 | 口径 | 依据 |
|------|------|------|
| 六边形朝向 | **平顶（flat-top）**，运行时换算与 DTLM 地图文件默认口径统一 | 2026-07-16 用户裁定 |
| 坐标系统 | 运行时按**轴向坐标**（`HexCoord` 的 Q/R）实现；是否定为最终选择仍在 [地图与移动 · 待确认事项](../../02-系统设计/02-地图与世界/地图与移动.md#待确认事项) 挂起 | 代码现状 |
| 格子尺寸 | 当前实现外接圆半径 **1 世界单位**；相邻格中心距 √3（占位值，未定稿） | 代码现状 |
| 地形颜色来源 | 初版**直接读取地图工具色卡**（DTLM 文件 `palettes` 中 contentId 对应颜色）；不在代码内维护第二套配色 | 2026-07-16 用户裁定 |
| 显示形态 | 占位地块（压扁圆柱）+ 每格颜色区分；不新增网格模型 | 2026-07-16 用户裁定（初版范围） |

## 模块组成

| 类 | 程序集 | 职责 |
|----|--------|------|
| `HexCoordinateUtility` | `Yanxu.Map` | 轴向坐标 ↔ 世界坐标换算（平顶、半径 1）；`AxialToWorld` / `WorldToAxial` |
| `MapGridHolder` | `Yanxu.Map` | 场景中持有 `HexGridStore`；新增 `LastImportResult` 保存最近一次 DTLM 导入结果 |
| `MapCellDisplayInfo` / `MapCellLayerEntry` | `Yanxu.MapView` | 单格显示信息：地形条目、其它图层条目、阻挡状态；条目含内容编号、标签与色卡颜色 |
| `MapDisplayModel` | `Yanxu.MapView` | 纯 C# 显示模型：以 `HexCoord` 为键管理全部格子显示信息；`Rebuild(HexGridStore, MapImportResult)` 重建；`Changed` 事件通知显示层 |
| `HexMapView` | `Yanxu.MapView` | MonoBehaviour 显示层：逐格生成占位地块，用 `MaterialPropertyBlock` 写入颜色；支持 `RefreshCell` 局部刷新 |
| `MapDisplayTests` | `Yanxu.Map.Editor` | 菜单自测（`Yanxu/地图/运行显示模块测试`）：坐标往返、邻居间距、模型构建与色卡颜色 |

程序集依赖方向：`Yanxu.Core` ← `Yanxu.Map` ← `Yanxu.MapView` ← `Yanxu.UI`；`Yanxu.Map.Editor` 引用 `Yanxu.MapView` 用于导入后联动刷新。

## 数据流

1. **原型场景**：`PrototypeSceneBootstrap` 初始化 `PrototypeContext` 后，用其 `GridStore` 构建 `MapDisplayModel` 并绑定 `HexMapView`。此流程无导入数据，全部格子按默认色显示（阻挡格用深色回退色）。
2. **DTLM 导入**：`MapImporterWindow` 加载地图文件 → `MapFileImporter.PopulateGridStore` 填充格子存储 → `MapGridHolder.SetImportResult` 保存导入结果 → 若场景中存在 `HexMapView`，重建显示模型并绑定，格子按色卡颜色显示。
3. **地形条目识别**：DTLM 图层名为「地形」的条目写入 `MapCellDisplayInfo.Terrain`；无该名图层时回退取首个图层。其余图层条目（环境、资源等）保留在 `Entries` 中供后续显示使用，初版不上屏。

## 颜色规则

| 情形 | 颜色 |
|------|------|
| 格子有地形条目且色卡命中 | 色卡颜色 |
| 格子无地形条目、不阻挡 | 默认灰（`defaultColor`，Inspector 可调） |
| 格子无地形条目、阻挡 | 深灰回退色（`blockedFallbackColor`，Inspector 可调） |
